# 上世纪最大骗局：毛岸英毛岸青是假的！ (2016-04-13 20:24:44) 下一个
（提示：这篇文章应该是一个系列，放在一起太长。但防止读者不能在一时间读到完整证据链，只好放在一起了。超长，慎入！）

毛泽东与杨开慧有三个儿子，大儿子毛岸英出生于1922年，13个月后二儿子毛岸青出世，到1927年又生了毛岸龙。由于毛岸龙下落不明，而毛岸英毛岸青宣称被找到了，根据官方文献记载，毛岸英死于朝鲜，毛岸青后来变成了傻子后与韶华结婚并育有一子毛新宇。

然而，根据润涛阎考证，所谓找到的毛岸英毛岸青二人并非真的是毛泽东的俩儿子。为了澄清谁谁是谁，本文用毛岸英、毛岸青代表毛泽东的俩儿子，而后来找到的所谓的毛泽东的俩儿子则用他俩自己给出的名字杨永福、杨永寿。

（一）关于毛岸龙的下落杨永福杨永寿与当事人的说法不同

据文献记载，关于毛岸英哥仨如何达到上海的，有三个版本。一个是：“1930年11月，杨开慧牺牲，湖南省党的地下组织将三个孩子护送到上海，经上海地下党组织安排，三个孩子进入董健吾牧师（当时是中共地下党员）任园长的大同幼稚园。”第二个版本：“1930年，杨开慧英勇就义后，毛岸英、毛岸青、毛岸龙三兄弟在党组织的安排下，跟着外婆、舅妈到了上海，来到叔父毛泽民、叔母钱希钧的身边。毛泽民将三兄弟安排住进了上海秘密党中央机关特科工作人员董健吾创办的大同幼稚园。”第三个版本：“1930年杨开慧牺牲后，毛岸英、毛岸青，毛岸龙兄弟三人就跟随杨开慧的母亲在家乡板仓生活。国民党在长沙地区到处抓捕共产党人和进步群众，毛岸英兄弟三人的安全也受到威胁。在上海地下党中央机关工作的毛泽民同志（毛泽东大弟）经请示党组织同意，将毛岸英兄弟三人接到了上海。”

虽然上面说法不同，但总体上来说是毛泽民安排哥仨去了上海，具体是谁护送的，与本文要探索的无关紧要。到了上海后的安排，所有的版本都比较一致：经毛泽民找到董健吾，毛岸英哥仨便被安排到了上海大同幼稚园。幼稚园表面上是一个收养社会流浪儿的教会组织，暗中却是共产党抚养革命烈士遗孤的福利机构。当时幼稚园的负责人是中央特科情报科成员董健吾，他对外身份是牧师。

（二）有关毛岸龙的下落

毛岸龙到底是跟俩哥哥流浪时走丢了，还是得病死在了大同幼稚园？在整个毛泽东时代，官方都说毛岸龙是跟着俩哥哥讨饭流浪时走丢了,导致党中央甚至毛泽东的亲戚比如贺子珍的妹妹等人一直到处寻找毛岸龙。毛岸龙失踪说均来自杨永福和他妻子刘松林、杨永寿和他妻子韶华：在1931年夏天，秘密党组织遭破坏，董健吾身份暴露，幼稚园的工作人员或被逮捕或隐蔽起来，孩子们因无人管理而流落到街头。毛岸英背着小弟，搀着大弟，三兄弟从此过着流浪儿的生活。毛岸龙突然发烧腹泻，毛岸英带着毛岸青出去乞讨，回来时毛岸龙就不见了。

上面由杨永福杨永寿说出来的毛岸龙丢失了的说法全国人民人人皆知。直到毛泽东死后由大同幼稚园当事人陈凤仙才讲出杨永福、杨永寿二人的说法是瞎掰，事实是：“1931年5月底或6月初的一天夜里，毛岸龙突然生病，腹泻、高烧，由我（保育员陈凤仙，又名秦怡君，李求实夫人）抱到附近的广慈医院就诊。医院诊断为噤口痢，经救治无效当夜死亡。次日由幼稚园负责行政事务工作的姚亚夫买棺入殓处理了丧事。”

电视剧《毛岸英》情景展示的是这一种说法。作为该电视剧总顾问兼总监制在接受专访时说：“医院的一位女同志曾对岸英说他弟弟得病毒性痢疾去世了。”

上面讲的“岸英”不是死在朝鲜的杨永福，而是毛泽东的儿子毛岸英。

润涛阎的解释：后来被找到的二人是湖南流浪乞丐杨永福和杨永寿，二人根本就没有三弟，也就在被追问毛岸龙去了哪里时谎称是走丢了。由于他们二人被毛泽东接纳为儿子，官方也这样报道毛岸龙的下落，抢救、下葬毛岸龙的当事人不敢在那阶级斗争残酷无情的年代里说明真相。而当毛泽东死后，尤其是胡耀邦拨乱反正人人可以说实话时才调查出来了毛岸龙的下落。从此共产党的宣传机构再也不承认毛岸龙失踪的说法可信。

2013年12月26日，是毛泽东诞辰120周年，中央决定开展隆重纪念活动。6月26日，韶山毛泽东同志纪念馆专题展区经过两年的重新调整，正式对外开放，共展出文物、图片600余件（张），其中不少珍贵文物系首次展出，背后藏着秘密故事，其中对毛岸龙的说明是：1931年初，毛岸龙由叔叔毛泽民安排进入上海大同幼稚园，不久因患痢疾去世。（参见《毛泽东爱子毛岸龙生死迷雾重重》）

润涛阎的解释：显然，杨永福杨永寿哥俩并不知道毛岸龙的确切死因，二人死前都一直说毛岸龙失踪了。表明杨永福杨永寿是假的毛岸英毛岸青无疑。

（三）流浪六年的杨永福杨永寿与毛岸英毛岸青的历史事实不符

根据后来官方的说法：“因顾顺章、向忠发先后叛变，使上海地下党组织遭到很大破坏，党组织在紧急隐蔽有关人员的同时，对大同幼稚园收养的多名烈士子女也进行了疏散。当时采取了三种疏散方式：有家可归的回家，有亲友的投奔亲友，无家又无亲友的由党组织托付给可靠人家。也就在这期间发生了一件令人痛心的事：1931年5月底的一天，毛岸龙突然生病，腹泻高烧，送医院后被诊断为紧口痢，经救治无效于当夜病亡。毛岸英、毛岸青属于第三种疏散的烈士子女，党组织决定把兄弟俩托付给董健吾抚养。毛岸英、毛岸青在董家住了一年半左右，并受到了良好的照顾，吃得饱穿得暖，有时还能看上电影。由于董健吾家‘松柏斋古玩号’是中央特科的重要联络点，这里不仅人来人往，而且距租界巡捕房又很近，党组织考虑到毛岸英兄弟俩满口湖南话，一旦遇到警察和特务盘问，后果不堪设想。便又将兄弟俩托付给董健吾的前妻黄慧光抚养（此事发生在1933年8月）。董健吾把毛岸英、毛岸青送到黄慧光家时隐瞒了他们的身份，只说这是他朋友的两个孩子，暂时住在这里。党组织每月给30块钱。后来，党组织没钱了，董健吾失去了固定收入，黄慧光是家庭妇女没有生活来源，自己有四个小孩，加上岸英兄弟一家七张嘴天天要吃要喝，日子过得艰苦了起来。生活艰苦，吃不饱肚子，还要挨打受骂。岸英自小就爱憎分明，一天，他看见岸青又挨了打，便离家出走了。”

而杨永福和他妻子刘松林先后提出二人流浪六年（杨永福的说法后被刘松林改成五年（刘在武汉的演讲）。刘松林曾说：“我们结婚后，岸英曾一次带我去西四胜利影院看电影《三毛流浪记》。当时，他触景生情，非常激动。电影结束了，影院里的人都走空了，他还沉浸在电影的情节中。他告诉我，他和弟弟在上海流浪的经历与三毛非常相似，说自己与三毛相比，除没有给资本家当干儿子和没有偷东西两点外，三毛所吃的苦他都吃了。回家的路上，他一直默不作声，他应该又想起了自己在上海不堪回首的6年流浪生活，我没有打搅他。后来，他也还曾带过其他人看这部电影，我想这与他个人的特殊经历有关。”

后来刘松林的该说法导致被告。2002年1月19日，刘思齐（=刘松林）应邀到武汉理工大学演讲。刘思齐在演讲中称：毛岸英和毛岸青在上海“像三毛一样流浪了五年”。对此，董健吾之子董龙飞认为，刘思齐的演讲中“有与历史事实不符之处，有些言论污蔑诋毁了董健吾，损害了董家人名誉，使其精神受到了损害”，逐向武汉市武昌区人民法院提起民事诉讼，要求刘思齐停止宣传与历史事实不符的言论，并登报赔礼道歉。

 董家后人提起民事诉讼的依据，来自于1979年中组部的一项调查报告。当时胡耀邦负责平反冤假错案，他指示中组部组成三人调查小组，专门赴上海调查了毛岸英兄弟三人在上海时的情况。通过访问当事人、证人和当时的地下党领导人，以及查阅档案资料，勘察毛氏兄弟居住过的地方，最后撰写了一份《毛岸英、毛岸青、毛岸龙在上海时的情况》调查报告，其中写到“……毛岸英和毛岸青在董健吾家中，生活虽困难，但无流浪、虐待事实……”。另外董家后人对“毛岸英兄弟流浪街头长达五年之久”的说法也提出了异议：毛岸英兄弟俩是1933年8月来到黄慧光家的， 1936年6月党组织送他们前往苏联，就算毛岸英兄弟俩刚到黄慧光家就立即出走，流浪时间最长也不会超过2年10个月。

润涛阎的解释：杨永福在看了《三毛流浪记》后深深感到那是自己和弟弟杨永寿的亲身经历，当场顾不得以前说出的谎言了，便说出了真相：他和他弟弟二人流浪了6年。如果他真的是毛岸英，那他就不会把在大同幼稚园和董健吾家一共三年多的美好时光忘记而把流浪二年十个月说成是流浪六年。

（四）杨永福杨永寿被强行说成是毛岸英毛岸青

当年“毛岸英““毛岸青”二人被找到的过程一直讳莫如深，连到底是怎么被找到的也无人得知。直到胡耀邦派调查组到上海查找有关毛岸英毛岸青毛岸龙的历史来龙去脉时，李云才出场说出她是当年找到毛岸英毛岸青二人的唯一之人。2011年11月29日，记者采访了当年中央特科在上海的最后一位成员——李云，才公开了许多鲜为人知的珍贵史料。对于当年毛岸英、毛岸青在上海失踪和寻找的过程，几十年来李云一直缄口不言，直到今天家乡来人采访，她才首次透露这段历史。根据老人的说法，毛岸龙也不是早年谣传的走丢了，而是在1931年“因病夭折”。董健吾告诉特科领导毛岸英毛岸青二人在董健吾的前妻黄慧光家由于不知为何毛岸青挨了打两个孩子就出走了，开始了流浪生活。特科领导立即布置地下党员分地区仔细寻找，但一无所获。

1935年秋，上海的地下党组织遭到严重破坏，中央特科成员大部撤离，只有少部分末暴露身份的成员留在上海。当时特科的总负责人是邱吉夫，主管情报工作的是李云的丈夫徐强（对外称老金，也是李云的直接领导）。一天，徐强突然严肃地通知李云，要她想办法上街寻找两个男孩子。孩子的身份对她也是保密的，只知道这两个孩子是烈士后代，大的十三四岁，小的只有十一二岁。

老人回忆说：“接到寻找两个孩子的任务后，我赶紧上街，可上海那么大，到哪里找？找了好些天，都没有发现他们，同时寻找过程中还要小心自己的身份不能暴露。一天，我一路寻找，后来走到城隍庙附近，想买碗粥填肚子。我看到粥摊边上有两个孩子，很瘦，穿的衣服也很破，年龄是符合的。旁边的人说，这两个孩子，从来不说话，很可怜的。我就请他们吃碗粥。吃后他们还是不开口，问什么都不讲。当时我也不敢多问，急忙回去向上海地下党领导人冯雪峰汇报了这个情况，并领着他们两人洗了个澡，换了衣服。冯雪峰立即换了便装赶来。冯雪峰来了后，连哄带骗带着两个孩子去了吃饭的地方，还不断跟他俩套近乎，可两个孩子很警惕。饭吃过后，大概跟冯雪峰熟悉了，冯雪峰问他们是哪里人，其中一个孩子才说了句是湖南人。毛泽东是湖南人，这下确定了两个孩子的身份。两个孩子找到后，董健吾通过张学良，让人于1936年6月带着两个孩子经法国去了苏联。到了莫斯科以后，他们被送到国际第二儿童院，各自取了苏联名字，开始了在苏联的生活。”

李云在此还特别强调了一点：她1930年入党，1932年被调到中央特科工作。中央特科是一个非常严密的组织，纪律非常严格，特别是情报工作。对上级的命令，必须绝对服从，不能问也不许问。她在徐强单线领导下，担任过报务员、交通员。徐强后来做了她的丈夫，并肩战斗了几十年。

原文来源：《大众文摘·解密（文史版）》

润涛阎的解释：李云并不知道让她去找的两个孩子是毛泽东的孩子，而被告知是烈士的孩子。她在此强调她不该知道的就没问。她天天在找俩孩子很久也没着落，又害怕暴露自己的身份，当她看到年龄看上去差不多的两个乞丐就带走了，交给了冯雪峰。文人冯雪峰唯一知道的信息是：这俩孩子是湖南人。显然，猫腻就在这里。李云不知道让她找的孩子是毛泽东的儿子。而在这里她说“毛泽东是湖南人，这下确定了两个孩子的身份”与她前边说的不知道让她找的是谁的孩子相矛盾，表明是她听到冯雪峰说让她找的是毛泽东的儿子后她才知道的。从此，她便害怕了，因为她显然听到冯雪峰问了那俩孩子的名字了。由于冯雪峰未留下有关俩孩子的只言片语，我们无法得知当时更全面的确切真相。但我们可以做出简单的合乎逻辑的推测：

这俩孩子是湖南人，年龄也跟毛岸英毛岸青差不多，就容易被冯雪峰认同他俩就是毛泽东的儿子。这俩孩子是很聪明的无疑，尤其是老大。经历了六年的流浪生活，阅历丰富。做梦他们也想有个富裕人家领养他们。当得知这善良女人不是简单行善给他俩吃的或领养他们，而是在替别人寻找孩子时，他俩必然清楚对方搞错了。这俩孩子的父母带他们从湖南来到上海做生意，被图财害命者给杀了，他们有幸跑掉活下来了而已。也许对手并不想做事太绝斩草除根。所以，他俩当然不回答冯雪峰的问话。他俩越是不说话，冯雪峰越是认为他们就是毛泽东的孩子，害怕给国民党诈出真实身份。而他俩的小加九则是：万一那个丢失两个孩子的家长找不到自己的孩子了说不定会领养我们呢！当李云得知组织上让她寻找的是毛泽东的孩子而非烈士的孩子后，内心忐忑不安。烈士死了，孩子是否是真的无法得知，也没人关心，可以蒙混过关。从她看到的两个孩子对冯雪峰的追问给出的眼神和表情等肢体语言，便猜测她找错了人。如果李云确定自己找到的真的是毛泽东的俩儿子，那可是天大的功劳，她李云不可能隐藏了一辈子此秘密，应该邀功才对。尤其是毛泽东打下天下后，她帮助毛泽东找到了俩儿子功勋卓著。毛泽东死前都不知道是她找到的这俩孩子。倒是冯雪峰邀功了，而且贪功之己有。毛泽东只知道是冯雪峰为他找到的俩儿子。冯雪峰与毛泽东在中央苏区时就是特别亲近的朋友，毛泽东后来对冯雪峰非常反感，世人不知道真正的原因，有不少人猜测毛泽东为何不喜欢冯雪峰，给出的理由五花八门莫衷一是。润涛阎今天告诉你：那是因为毛泽东认为冯雪峰在俩个儿子上欺骗了毛泽东。连俩孩子的真实身份都没搞清，至少是对毛泽东的不尊重。冯雪峰从1954年开始就遭受批判，后来的遭遇更惨。大家都知道这是毛泽东故意找他的茬造成的，至于为什么，人们不清楚。

根据润涛阎的推测，冯雪峰并没有故意欺骗毛泽东，他见到李云给他找到的俩湖南乞丐就如同座山雕见到了朝思暮想的联络图，理所当然认为李云给他找到的就是毛泽东的俩儿子。冯雪峰为了让毛泽东的俩二子早一刻喜笑颜开，便告诉他俩：你爸现在是共产党的最高领袖。他俩听后自然害怕，骗一个普通的失去儿子的富豪都办不到，只能盼望被领养而已，可是要骗领袖级别的大人物，那是找死，便战战兢兢，宣称自己是杨永福而非什么毛岸英。他俩越是害怕，冯雪峰越是认同他们就是毛岸英哥俩。可当他俩得知他们不是去延安见丢了俩儿子的毛泽东，而是直接去苏联，便开始配合了。反正是走一步看一步，对乞丐来说，能到外国留学、生活，那是天上掉下来的馅饼。他俩此时的心理便倒过来了：由“害怕说成是大人物毛泽东的儿子而遭受惩罚”变成“害怕被发现不是毛泽东的儿子而被遗弃”。走出这一关而不被察觉的关键是不说话，话多语失。但听话，听从安排，事事都点头。这样，在没与毛泽东见一面的情况下，也没有问出他俩的名字是不是毛岸英毛岸青就稀里糊涂地把他俩送到了国外。真实情况只有到了1936年他俩回到了延安才被毛泽东发现。

（五）真实的毛岸英毛岸青最可能的下落

简单回顾一下上面提到的毛岸英毛岸青的简历，其实过程并不复杂。毛岸英哥仨是1931年初到了上海大同幼稚园，此时一个九岁一个八岁一个四岁（取整数）。不到半年毛岸龙得毒性痢疾死在了医院（五月底六月初），埋葬时有棺材，丢失一说是瞎掰无疑。顾顺章是四月底被捕，向忠发是六月22日被捕24日被杀。显然上海特科组织疏散发生在顾顺章被捕之后和向忠发被杀之前，也就是五六月份。大同幼稚园一直开到六月初。此后毛岸英毛岸青哥俩便去了董健吾家，在那里生活了一年半到1933年8月，董健吾把他俩送到董健吾的前妻黄慧光那里。

在上海特科人的眼里，周恩来是他们的老大。从遵义会议后周恩来便对毛泽东非常尊重，董健吾不仅仅是看在周恩来的面子上对毛泽东的仨儿子特别关照，还有另外的一个原因：周恩来给照顾毛泽东儿子的人每月30块现大洋。在日本进关以前，清华北大学生每年的学费也就是20到30块钱。要知道，毛泽东在北大当图书管理员时的工资是每月八块钱。董健吾愿意留养毛泽东的两个儿子在自己家中，敢冒被抓被杀风险，也有每月30块钱的物质诱惑。后来万不得已才因只会讲湖南话的毛岸英二人留在家里风险太大而把他俩转到前妻那里。

前妻黄慧光为何会收养毛岸英毛岸青二人？

黄慧光是在董健吾与一同革命的女同事勾搭成奸后被遗弃的，也有黄慧光担心害怕丈夫当地下党被连累的因素才答应与董健吾离婚。离婚后的4个孩子都跟着黄慧光，董健吾把毛岸英毛岸青让黄慧光收养，这30块钱也让黄慧光给他生的四个孩子有饭吃。等于肥水不流外人田。黄慧光自然接受这样的好事。可到了董健吾拿不到中央发下来的钱后（中央特科整个系统在上海已经支离破碎，活着的也互相不联系了，害怕被抓），黄慧光无法接受白养毛岸英毛岸青了。这就有两个选择：一是打他们，最终逼他俩滚蛋。但这有风险，说不定这俩孩子会报复她。另一选择就是把他俩换成钱。也许一开始黄慧光走的是第一个选择，等到他俩被打后满脸怒气【毛泽东的爸爸是暴脾气（毛泽东自述的），毛泽东也是不吃亏的主，那毛泽东的儿子们也不是窝囊废一级的软柿子。】此时黄慧光发现逼走他俩的后果非常危险，便联想到自己被女共产党员小三换位、被男共产党员丈夫抛弃，对共产党的仇恨那可是深入了骨髓。这俩孩子不知道是哪个共产党的血脉，但一定是共产党的后代无疑，董健吾的大同幼稚园就是专门抚养共产党烈士后代的机构，反正这俩孩子的父母都不在世了，没必要为他们俩担当被政府抓的风险。

当时国民党在顾顺章叛变后捣毁上海特科时发布布告，凡是隐藏地下共产党的格杀勿论，凡是举报共产党和他们亲属的有偿。出于害怕被牵连、出于对共产党的仇恨、出于对卖掉共产党烈士的后代有偿，三个因素的其中任何一个都能让黄慧光把毛岸英哥俩出卖给国民党换成钱，何况三个因素都凑齐了。国民党晚上去她家抓走俩孩子然后杀掉，由于不知道他们是毛泽东的后代，也就给点小钱而已，也不会上新闻，毕竟杀死孩子有点恶毒，就像杀死小萝卜头被曝光导致很多文人痛骂国民党蒋介石一样。所以，毛岸英毛岸青应该死于黄慧光拿不到共产党的钱了后不久的1933年。

董健吾去黄慧光处查看毛泽东的俩孩子时黄慧光自然不会告诉他是她出卖了毛岸英毛岸青，而是说她没钱买粮食养他们俩，因为打了老二，老大就带着他出走了。董健吾肯定害怕了，他知道那俩孩子是毛泽东的后代。尤其是此时毛泽东在延安已经是权高位重了。他当然能判断出最大的可能是黄慧光这个恨共产党、见钱眼开的短视妇人把毛岸英毛岸青出卖给了国民党换了钱。但他不能这样告诉共产党上级，因为毛泽东的孩子是交给他照顾的死了他有责任，何况那女人还是他的前妻。他只能告诉上级说毛岸英毛岸青出走去流浪了。上级立刻派人在上海四处寻找。三年时间内把上海大街小巷搜查了无数遍也没发现毛岸英毛岸青的踪影，但根据周恩来的指示，寻找一直没有间断，李云是最后一拨。理论上讲，杨永福杨永寿哥俩在六年的流浪生活中的最后两年多时间里应该被寻找毛岸英毛岸青的地下党特科人员询问过。只是碰到的都是负责任的人，最后才碰到了怕暴露自己身份而敷衍了事交差就行的李云（她认为反正就是烈士的俩孩子，错了也没什么大事何况烈士是死了的人，其他人无法辨认出来，乞丐本身更不会说出真相放弃前途而继续流浪生活），也碰上了个书呆子诗人急切邀功的冯雪峰，才有了后面精彩的故事。

（六）杨永福杨永寿回到延安

在苏联期间，杨永福杨永寿以毛泽东的孩子身份处处被照顾，但也算是被监督，到哪里去、能否留在苏联都不是他们自己能决定的。1946年党组织要他俩回国，也许是毛泽东的命令，他俩就必须服从，不论心里多么忐忑不安多么担心害怕。

杨永福杨永寿哥俩以毛岸英毛岸青的名义回国到了延安后，最想见他们的是毛泽东无疑。毛泽东在1927年底去上井冈山离别时毛岸英5周岁（取整数，4.8 或5.2 岁在这里没必要），毛岸青4周岁。此时再见面哥俩已经是19年后的24岁和23岁的成年人了。如果论孩子的貌相没有毛泽东杨开慧的蛛丝马迹而判断此二人属于狸猫换太子，可靠性不是百分之百，毕竟人到了老年才有与父母不可分割的印记（此话题后面有更详细的交代，在此按下不表），但毛泽东毫无疑问会问及小弟弟岸龙的话题。由于杨永福杨永寿哥俩只知道他们代替的是毛岸英与毛岸青，从未听说过还有个小弟弟毛岸龙，俩人突然就懵了，本来在苏联的多年时光里哥俩就探讨未来不得不回到中国去见毛泽东时如何才能躲过毛泽东的盘问，可万没想到还有个毛岸龙的事。聪明的杨永福当即给出了个说法：“每天背着小弟拉着大弟流浪乞讨，有一天岸龙突然发高烧，我就带着岸青去乞讨，回来就发现小弟不见了。”

毛泽东是什么人？他如何能被一个年轻人骗过去？察言观色他便得知是冯雪峰搞错了，把两个会说湖南话的乞丐当成了毛岸英毛岸青给送到了苏联。此时的杨永福杨永寿毕竟还做不到面对突如其来的处境而喜形不于色。

那毛泽东为何不把当事人冯雪峰叫过来把找到俩孩子的来龙去脉搞清楚？

因为毛泽东一直在庆幸自己没断子绝孙，这对于天天看线装书的毛泽东来说是至关重要的，至于是不是他要让他的孩子接班成为北朝鲜的金家并不重要，重要的是他不能被敌人骂成断子绝孙。他整自己人无数，杀敌人无算，被人骂成断子绝孙是比被敲脊梁骨还难受的。当他知道那俩孩子不是他的儿子后，便将计就计，就坡下驴，认了这俩儿子。即使以后能找到真的毛岸英毛岸青，到时候再给出说法不迟，就说我毛泽东虚怀若谷，早就知道真相照样把穷人的孩子当成自己的儿子看待。这更令人崇拜其高尚品德。何况那时周恩来等人都领养了烈士后代，有儿女的叶剑英邓子恢等人照样领养了烈士遗孤。在毛泽东看来，不就是吃饭时多两张嘴？反正饭也是公家的。

对于毛泽东来说，关键的一点是他不允许别人骗他。根据这个原则，他理应把冯雪峰叫到眼前询问，结局当然会立刻水落石出。但这对毛泽东来说没任何好处。他清楚那个冯雪峰就是一个诗人，充满了浪漫情怀，又是自己的好友，绝对不是故意骗他的。最可恶的是这俩孩子竟然知情不报。估计毛泽东权衡利弊后觉得还是借刀杀人比较好。可派他俩去前线打仗，哪个军头也会把他们保护起来。倒是在农村有不少地痞流氓，让他俩去农村参加土改，说不定这事就成了。于是毛泽东便跟他俩说：“你们在苏联长大，中国的事你们不太了解，需要到下面去锻炼。你们是毛泽东的儿子，更需要到基层，亲自参加土改，对你们的成长有帮助。”这就是在1947年初杨永福去了山东搞土改，杨永寿去了东北搞土改的过程。不论山东还是东北，都是土匪聚集的地方。带他俩去的人都有极左性格。比如派带杨永福去山东搞土改的就是左棍康生。可这哥俩命大，都没碰上要他们命的，表明他俩很聪明。据报道，去黑龙江克山县搞土改的杨永寿在那里纠正了左倾错误，导致了不少人免于被杀。他还用中文自编了多首土改民谣。

杨永寿在苏联学的是俄文，他的中文底子应该是在上海时经商的他爹给他俩放入私塾学堂学的中文。这兴许是他俩没跟父母一起被杀的原因。他俩在学堂住宿逃过了一劫。待没人给他们交学费他们才被学校撵走从此流浪六年后被寻找毛泽东俩儿子的李云误认。杨永福杨永寿的确切年龄我们无从得知了，兴许比毛岸英毛岸青大一点也是可能的。这就解释了他的中文回国后还能编写民谣的水平（有相反的报道，说他俩在苏联时不会用中文书写，只好用俄文写好信后由朋友帮忙翻译成中文再寄回中国）。对比真的毛岸青，只在大同幼稚园待了几个月，可以学点学龄前的一点点算数语文。几个月学不了多少字。后来就在干革命的董健吾夫妻家中，董健吾的亲人只说他俩在董健吾家有吃有喝，有时董健吾夫妇带他们去看电影，没说雇人教他们学习。董健吾的前妻是个家庭妇女，每天照顾六个孩子，董健吾的亲人也没提他俩在黄慧光家有人教他们学习。从这方面看，杨永寿能编写土改民谣如此扎实的中文底子不是真的毛岸青所能具备的。

天下打下来了，杨永福杨永寿也就回到了北京，入住中南海。骗了毛泽东的杨永福杨永寿哥俩土改这一关就这么平安过去了，令毛泽东内心里失望之极。反观杨永福杨永寿哥俩，误以为真的骗过了毛泽东，从此自信心爆棚。

（七）杨永福大胆抢毛泽东的干女儿

烈属寡妇张文秋有三个女儿（见下图）。



左边的是所谓的“毛岸青”，最右边的男人是刘思齐改嫁的丈夫。三个女人分别是：最左边仔细端详毛泽东的是三女儿，毛泽东另一边的是大女儿（刘思齐）和二女儿（韶华）。

张文秋曾经是毛泽东的邻居，她把她大女儿刘思齐托付给毛泽东当她的干爹。从此，干女儿刘思齐就跟在干爹毛泽东身边。尤其是江青因为发现毛泽东的这个干女儿总是往毛泽东那里跑而吃醋。杨永福对毛泽东的这个干女儿一见钟情，对江青指责毛泽东认干女儿非常恼火，曾在毛泽东面前怒斥江青。后来毛泽东得知杨永福在追干女儿，久而久之二人还谈起了恋爱，并提出要跟刘思齐结婚。毛泽东大怒，坚决反对杨永福干涉毛泽东与干女儿的关系，不允许他俩结婚。

理论上推理是刘思齐跟干爹谈：你不能跟江青离婚，虽然你告诉我说她得了卵巢癌后手术割掉了卵巢而丧失了性欲，但你跟我说你不能跟她离婚。那我跟你儿子结婚，我是你名义上的儿媳妇，你可以派他去外地工作，把我留在家中。这不是更能掩人耳目吗？毛泽东这才答应了刘思齐可以与杨永福结婚。毛泽东与杨永福大吵大闹不让他与干女儿刘思齐结婚的事，整个中南海人人皆知。闹得毛泽东也不好意思继续大骂他了，只能找机会让他为国捐躯了。

杨永福去朝鲜前并没有告诉新婚妻子他要去哪，显然是毛泽东给他的建议别打搅她。有文件披露说是杨永福托人告诉妻子他出差了。当他跟彭德怀去了朝鲜后，按照毛泽东给他的教导，他从未给新娘子写过信交代一下他在朝鲜。他死后的差不多三年时光里，他法律上的妻子从未跟她也住在中南海的干爹打听过丈夫出差去了哪里，为何近三年都没给她写过信。直到新婚夫妻分手两年八个月后的一天，她才问起干爹“毛岸英”（=杨永福）去了哪里。干爹才告诉她他去了朝鲜一个多月后就去世了。干女儿听到这里才在干爹怀里大哭了起来。她内疚是自己配合干爹害死了与她毫无恩怨并深深爱过她追求过她的乞丐杨永福。在后来的访谈中，她刘思齐说起这件事时是这样解释的：“我那时年轻，还不懂得丈夫走了三年妻子应该过问一下他去了哪里、为何三年都没给我来信。”

就学于莫斯科大学、北京大学的女人、毛泽东的干女儿竟然不知道丈夫走了三年不给她写信她需要跟身边的公公打听一下到底发生了什么事。别说能读大学的女人了，就是不识字的白丁，新婚丈夫出差别说差不多三年没消息，就是30天她都得惦记着，就得跟身边的公公询问是怎么回事。就算丈夫出远门，30天妻子放心，三个月沓无音讯，新婚妻子又没有跟其他男人出轨，婚姻是真实的，那她能不急得跳脚？我们结过婚的男男女女，大家扪心自问：这等婚姻是正常的婚姻？丈夫只告诉妻子出差，不告诉去哪里，本就对新婚妻子不尊重。什么政治啊，斗争啊，把新婚妻子仍在家自己出远门还不告诉她去哪里，妻子怎么想？天下都打下来了，丈夫也不是去外国当间谍，就算是出国当间谍，别的人不告诉怕走漏风声，有何必要隐瞒新婚妻子？这是对妻子的不信任。走了快三年沓无音讯，妻子不过问原因。如果你的脑袋没被门夹过，你绝不会把刘松林真的与爱着他的丈夫接了婚的事当真。这事如果发生在美国，那移民局一定判断出是假结婚拿绿卡的所谓的夫妻。

得知杨永福死在朝鲜后，这新婚寡妇在中南海跟毛泽东又生活了八年才改嫁，把人生中最美丽最美好的青春埋葬在了中南海厚厚的红墙内。在这漫长的八年时光里，她每年夏天都跟毛泽东去北戴河。据罗瑞卿的女儿罗点点撰文说，刘思齐在北戴河只跟毛泽东谈话，其他人她理都不理。

显然，得知杨永福死了的消息后，刘思齐极大可能得了抑郁症。她变成了个谁都不理的孤独女人，有她自认为中南海人人知道她的秘密而无脸见人又不得不出来放放风才到北戴河散散心的原因。

（八）彭德怀雪夜拜访金日成后搞清楚了天大的秘密

历史上曾经发生过一些令后人难以置信的真实故事，其一就是彭德怀猛扇金日成耳光，而且是在金日成手下人和彭德怀手下人面前扇的，金日成想隐瞒被扇耳光的事都办不到。

对此，在场的人不得要领，便猜测是在第五战役期间金日成为了保存自己的实力而私自下令北朝鲜军队撤出战斗惹怒了彭德怀。而不信彭德怀会扇金日成耳光的人找到了证据：金日成根本就不知道彭德怀的第五战役作战计划，彭德怀也没让金日成的部队参战，从而判断出彭德怀扇金日成耳光是无中生有。然而，在场的人不止一个承认了彭德怀扇金日成耳光的事实。在庐山会议彭德怀倒台被批时，“不尊重友邦、搞大国沙文主义”便是彭德怀的一大罪状，说的就是他扇金日成耳光的事。彭德怀对此没有提出反驳。

1955年金日成曾到中国与彭德怀见过面，跟外宾见面时要有其他外交人员或翻译在场。金日成与彭德怀举杯，表面上看是金日成原谅了彭德怀，而事实上是金日成想找机会跟彭德怀聊聊家常，告诉他他不知道的内情。

金日成终于找到了这样的机会，那是1957年11月2日彭德怀陪同毛泽东访问苏联，在11月16日，彭德怀得到了毛泽东的同意去见住在苏联别墅休假的金日成。彭德怀没有带身边的王树声、肖华等军人一起去，只是带了俄文翻译孙立忠。据后来孙立忠的文章说，彭德怀带他在去见离莫斯科40公里开外的金日成，说彭德怀与金日成谈了两个多小时。但孙立忠没说他也参加了彭德怀与金日成二人的会谈。金日成是在中国长大的，讲一口流利的中文，自然不需要翻译，何况孙立忠是俄文翻译，不懂朝鲜语。显然他只是陪同彭德怀去的，到了会见时他就在大厅里等着，这更符合常理，毕竟他只是个年轻翻译。孙立忠从未提及彭德怀与金日成密谈的内容。

从东方文化角度来讲，金日成与毛泽东都是国家级领导人，按共产主义运动阵营里的排位顺序此时应该是斯大林第一，毛泽东第二，金日成与胡志明比毛泽东低一点，但还是属于同一个级别的。如果说斯大林毛泽东金日成属于专制社会的皇帝，彭德怀就是大臣。按理说金日成也在苏联，中朝两国领导人会面，应该是毛泽东与金日成二人。彭德怀为何敢越位去见金日成？显然是金日成派人跟彭德怀联系上了，彭德怀也想借机给金日成道歉，因为扇耳光后金日成到北京时对彭德怀非常尊重。据孙立忠描述，当他们的车到达金日成别墅时金日成已经在寒风里提前出迎了。下面是连孙立忠都不知道被金日成彭德怀二人至死没留下笔墨的金日成与彭德怀密谈的内容，由政治家们肚子里的蛔虫润涛阎提供。

二人寒暄落座后彭德怀先给金日成道歉，说自己脾气太坏，加上当时的第五战役太悲催，就跟您老动了手，一直内疚。今天有机会给您道歉，倍感荣幸。

金日成听后皱起了眉头：我还以为你彭德怀清楚了事情的原委，现在看来，彭总你还是闷在鼓里！你为何扇我耳光？那是因为你在朝鲜的志愿军司令部（简称志司）换过六个地方，挨过美国空军九次轰炸！你找不到原因，便认为是我给敌人通了情报，目的是借美国的炸弹杀死你，因为你不尊重我金日成，我杀死你后毛泽东便会派林彪来指挥志愿军。我在东北联军时最佩服的是林彪，而与你彭总没接触过。林彪打仗我佩服，而且他会更尊重我金日成。你这样判断是合理的。尤其是毛泽东的儿子被炸死后，你非常痛苦，因为你知道我并不清楚毛泽东的儿子在你身边，那我想炸死的不是毛泽东的儿子而是你彭德怀，你便对我恨之入骨，找到机会扇我耳光，一是报复，二是对我警告以后别再把你的地址通过间谍告诉美国人。你说我猜得对不对？

彭德怀一听愣了！这金日成与我彭德怀一样有话直说啊！佩服！然而，彭德怀不知如何应对对方的坦诚，只能皱眉头，也从内心里佩服自己当年的判断是准确的。

但听金日成来了一句：彭老总，你的猜测是错的！我当时的确不知道毛泽东的儿子在你身边，但暴露给美国空军你们志司所在地的不是我，而是你们中国人自己。

彭德怀又愣了。彭德怀早就把身边知道志司所在地的人一个个都仔细过滤过，没发现有这样的人。便问金日成此人是谁。

金日成反问彭德怀：

“在11月24日前是不是美国的侦察机去志司那里侦查过？”

“对啊！23日那天就在我办公的房子上盘旋了一阵子后我们才决定以后白天要到防空洞躲避。”

“那你想想，那阵子你和北京是否有很频繁的电报往来？”

“是的！我们研究好了的第二战役作战计划电报发给毛泽东后他改了很多便打过来，我们再研究然后又发过去，他又打回来提出很多建议。所以，来来往往电报很频繁。”

“这就对了！毛泽东为何管你那么仔细？你想想看，他能不知道美国侦察机能准确定位电报发出地？”

“这个我当时就反复想他怎么那么啰嗦，这么长这么频繁的打电报如何不让美国情报机构测到？可您老别忘了，我身边有毛泽东的儿子！所以，即使是电报导致的被美国侦察机发现了我们的住处，那只能说毛泽东太大意了，对作战计划的重视超过了对他儿子性命的重视。”

“哈哈哈！我说老彭啊，你到今天都不知道毛泽东为何要抗美援朝。你听我说说我的看法对不对。”

“您讲。”

“毛泽东抗美援朝的目的有四个。在讲这四个目的之前我需要问你一件事，我在中国的时候就听说毛泽东的三个儿子都成了乞丐，后来找到了两个，老大和老二。他俩年龄只差一岁。后来找到的这两个你都见过，凭你的直觉，找到的这两个看上去也是只差一岁？还有，跟着你的老大他看上去有毛泽东的痕迹？一句话，那俩孩子真的是毛泽东的俩儿子？”

“老金啊，您这玩笑开大了！中国共产党员里哪里有人如此大胆敢骗老毛？”

“老彭啊，我只是提醒你一下，你自己以后慢慢想。我是判断出来了，那俩孩子根本就不是毛泽东的儿子。”

“老金啊。我还真的怀疑过为何这毛岸英相貌上一点老毛的痕迹都没有不说，他弟弟就更差的离谱了。他俩相差至少三岁以上，而不是只差一岁。这毛岸英的气质性格与老毛也是没一点相似之处。我也自言自语过，老毛怎么会有这么个儿子呢？可您要知道，这俩孩子是老毛当年的好友、没心机的文人冯雪峰给他找到的。是他亲自写信给毛泽东邀功帮他在上海找到了他俩儿子。退一万步，就算冯雪峰搞错了，那老毛的其他儿子也都死了，就算领养俩儿子有何不妥？他为何要借美国轰炸机把他杀掉？”

“这不难理解。毛泽东把自己的儿子还是长子都送前线成了烈士，那毛泽东的威望就更高了，他就更容易一辈子当政啊。还有，领养是领养，讹骗是讹骗。毛泽东怎么能容忍别人骗了他？尤其是毛泽东发现这俩孩子自己清楚是骗了毛泽东，他俩要是在延安第一次见到毛泽东时就说出真相说哥俩是被动的，怎么解释冯雪峰都不听，那毛泽东就不会怪罪他俩了。还有一个原因：毛泽东的儿子死的死丢的丢，他害怕被对手骂成断子绝孙，也就忍下了这口气。等到有机会再报复他俩不迟。”

“佩服！您是能钻进毛泽东肚子里的人啊，判断的非常有道理。有一点您不知道，在老毛让这孩子跟我来之前，他们爷俩因为一个女人而打架，老毛大骂，而他也大声吵闹，中南海人人皆知。后来老毛同意了他跟这个女人结婚。她本来是老毛的干女儿，在老子身边晃荡时被儿子一见钟情了。他要是不跟老毛大吵非要娶老毛的干女儿，老毛也许就对他不告诉老毛他不是毛岸英而原谅了他也说不定呢。”

“老彭你这一说就更与我的判断吻合了。这是一个目的，另外三个目的......”

“老金啊，我知道另外三个目的分别是：借机炸死假毛岸英时也把我炸死。如果他儿子不来我身边，通过跟我没完没了地电报联系而借美国之手炸死我，一定有人会怀疑老毛是故意借刀杀人。有他儿子在我身边我俩一起死，就不会有人怀疑是老毛故意借刀杀我了。第三个目的是杀掉林彪的兵马。虽然是我当总司令，可志愿军里的大多数军人还是林彪四野的。林彪的四野号称百万大军，老毛需要给他瘦身。而且，四野里哪部分来，也是老毛敲定的。里边国民党降卒比例很高。老毛对国民党投降的官兵是不信任的，因为他们没经历过共产党的政治教育。老毛也看不起国民党的官兵。最后一个目的是他跟我当面讲的，他说：‘我们一出兵，斯大林就对我们不怀疑了。’是斯大林让他老毛出兵朝鲜的。”

“老彭厉害！我一直以为你是带兵打仗的，对政治不在行。今天看来我错了。还有一件事，就是你知道为何毛泽东对美国两次提出的停战谈判态度大相庭径？”

“今天听您老这么一说我都明白了。到了第五次战役后美方给出的停火条件远远低于第二次战役后美方给出的停火条件。第二次战役结束时我们已经收复了三八线，老毛不停战可他也没有能力解放南朝鲜，我一直不理解他的用意。到第五次战役后，还是在三八线。显然，在第二次战役后，老毛抗美援朝的四个目的只实现了两个：一是斯大林满意了，二是炸死了他一个假儿子。到第五次战役后，林彪的四野死掉的达到了老毛的预设目标了，唯一遗憾的是我彭德怀还活着。他觉得我的命硬，再来九次轰炸我还是死不了。所以，他就认了，对于停战条件，他不怎么感兴趣。”

“看来老毛这人怎么想的，我们只要稍微一动脑便可明白。”

“您老先接受我的道歉吧！我就是没有您的政治头脑才误判的，才扇了您耳光。我现在追悔莫及啊。”

“老彭啊，这你就错了！我感激你的那个耳光啊。没有那个耳光，我绝不会在美国不撤出南朝鲜的前提下我就把志愿军全部轰走。我做那个决定就是想到你那个耳光，而且是在我手下人在场时扇的，在我脑子里你不是彭总司令，而是代表着中国。现在看来，我当初的决定是对的，我把延安派彻底从我的班底扫除干净就避免了给中国当藩属。正是因为你的耳光彻底断绝了我与中国人真心友好的想法，我才有今天与毛泽东平起平坐的地位。这都是你老的恩赐，我才跑到这里跟你谈我的真心话。当然，我清楚你就是死到临头也不会出卖朋友，我才敢跟你交心。”

“这一点您说对了。就是老毛再找机会弄死我，我宁死也不会出卖您老对我的真诚相待。”

彭德怀在庐山会议上天天纳闷，为何毛泽东还不动手？我配合你干掉刘少奇，否则我不是白白得罪邓小平了不说，你老毛要单独背上大跃进饿死人的历史罪名。等到会议快要结束了，彭德怀等不及了才写信提醒毛泽东：大跃进饿死的人太多了。他误以为毛泽东不知道饿死那么多人。在长信中毛泽东能看出：彭德怀把大跃进正确的政策（毛泽东定的）与执行中的错误（刘少奇邓小平搞的）区分开来了，等于告诉毛泽东饿死人的罪恶可以推给刘少奇。毛泽东终于等到了彭德怀跳了出来。在毛泽东眼里，彭德怀是要当大英雄的，当年不经毛泽东批准就擅自搞了个百团大战，把抗日英雄的帽子戴在自己头上而毛泽东反对彭德怀打日本、只打内战、发展自己军队不抗日的内心暴露无遗。抗美援朝，彭德怀高兴极了，他又有机会当大英雄了。现在大跃进饿死人了，是毛泽东刘少奇干的，彭德怀一定会出来“为民鼓与呼”。毛泽东当即在彭德怀与刘少奇等多人面前挑拨离间说：你彭德怀能不能在我死后不造反？我死后谁还能管得住你？这就是让刘少奇死战彭德怀。

毛泽东清楚彭德怀信中说的内容是真实的。大跃进饿死那么多人而毛泽东要是颠倒黑白倒打一耙说彭德怀是反党分子，其危险还是不小的。开会的人心里都有一杆秤，都清楚彭德怀是对的，而打倒彭德怀是违反人心与天理的。所以，毛泽东亮出了两张牌，一张是“军队不跟我走我就去井冈山”---恐吓牌，另一张是我无后了，儿子死的死疯的疯---悲情牌。甚至说出自己“应该断子灭孙。（毛泽东原话）”，这两张牌一出手，毛泽东又赢了，彭德怀输个精光。毛泽东在此提到他无后了，断子绝孙了，还有另一用意：你彭德怀别用历史罪名圈我入套，我断子绝孙了，不在乎历史名声也要把你干掉。虽然他口头上这么说，但他心里还是惧怕他死后遭鞭尸的。在对彭德怀的审查期间，刘少奇建议对彭德怀判处死刑，报告打到毛泽东那里，毛泽东没答应。第二次还是建议彭德怀以“里通外国”罪判处死刑，毛泽东还是没签字，等于没认可。

毛泽东反复领教了彭德怀的命太硬。在井冈山时期有一次彭德怀的第三军团被国民党围杀几乎全军覆没，眼看他自己也要被活捉便大喊身边的人一起跳崖，为共产主义流尽最后一滴血。当他跳崖后醒来发现自己没死而是挂在了一棵大树杈子上。这棵大树只挂住了他一个人。他想自杀都死不成；胡宗南十万大军杀不了他；百团大战时彭德怀的指挥部离日本军队最近时只有500米远，日本人杀不死他；在朝鲜美国轰炸机9次成功轰炸了他的住处，没伤到他一根毫毛。彭德怀当上了抗日抗美民族大英雄，大跃进饿殍遍野，彭德怀站出来“为民鼓与呼”，再次在历史上留下了大英雄的重重一笔。此时谁杀了彭德怀，谁就是历史上第二个秦桧。刘少奇想当秦桧，毛泽东倒不是不想当赵构，而是他清楚彭德怀的命太硬，杀了他自己可能也得很快被阎王爷带走，他也不想成全视死如归的彭德怀当上历史上的第二个岳飞。而且毛泽东公开杀死彭德怀，后人在彭德怀庙里给彭德怀铸的铜像前边下跪的毛泽东刘少奇二人铸像用的则是黑铁。门口的对联只改一个字，便是：青山有幸埋忠骨，黑铁无辜铸罪人。所以，不论毛泽东对彭德怀“刚愎自用，目空一切（毛泽东原话）”多么恨之入骨，他也不敢公开杀死彭德怀。

上面用如此长的篇幅写彭德怀，就是因为彭德怀的命运与杨永福的死分不开。如果金日成没机会私下里密会彭德怀，毛泽东不怀疑彭德怀知道了毛泽东要借刀杀了他，也就会在庐山会议上利用彭德怀等人的力量把大跃进的罪责栽到刘少奇的头上，以后有机会再整彭德怀不迟。历史发展过程会改写，但专制制度下的绞肉机互相残杀是无法停止的，跟随毛泽东打天下的功臣们的悲惨结局是注定不会改变的，改变的只是时间顺序。

（九）毛泽东一句话吓傻了杨永寿

杨永福死在了朝鲜，与他是否吃蛋炒饭关系是怎样的？如果他不吃蛋炒饭，兴许他也跟着他人去防空洞而死不了。不过，那只是暂时的，他的命已经掌握在毛泽东手里。无论如何“做蛋炒饭烟囱冒了烟而被敌机发现”的说法站不住脚。北朝鲜大部分是山地与丘陵，农民零零散散地生活在山区，到处都是冒烟的农家烟囱，要是有冒烟的房子都炸，那不知炸死多少人后才轮得到炸死杨永福和高瑞欣。事实上，早在前一天美国侦察机就捕捉到了志司的电报机频繁地发出的信号，不论第二天烟囱冒不冒烟，给总司令部扔凝固汽油弹是必然的。

杨永福死后，杨永寿一直被埋在鼓里。当他得知他哥哥被炸死在朝鲜，他当即找毛泽东发怒，因为那是他唯一的亲骨肉，而且是哥哥带领着他流浪六年，一步步把他拉扯大，生死与共，其恩情超过一般人的父爱。杨永寿清楚，是因为他哥哥跟毛泽东争毛泽东的干女儿才导致被毛泽东借刀杀死的悲剧，他的满腔怒火难以压制住。毛泽东待他发完脾气后问他：你们哥俩骗我，不告诉我实情你们并不是毛岸英毛岸青，在古代这是欺君之罪。你以为我就那么好骗？你们俩乞丐竟然想玩弄我毛泽东？

杨永寿回复道：我们哥俩没骗过任何人，也没骗过您。当时我们哥俩怎么跟冯雪峰争辩说我们不姓毛，我们不是什么毛岸英毛岸青，我们是杨永福杨永寿。可我们说不服他。

毛泽东愤怒地质问道：你说这些我相信是真的，但你们回国到延安与我第一次见面时为何不告诉我实情？我问你们岸龙怎么回事，你们当即就慌了。从那一刻我就知道你们俩是在骗我。我俩儿子只差一岁，你们俩差多少岁？如果你们俩任何一人说出真相，我也不会怪罪到你们头上，也会收你俩为养子，对外就说是我的俩亲儿子。可你们耍小聪明，想骗过去！

杨永寿道：您错了！我们俩害怕的是一旦告诉您实情，冯雪峰就倒霉了，我们不能恩将仇报。没有他，我们俩就得继续讨饭。所以，就只好支支吾吾了。

毛泽东道：你现在还在找理由狡辩。你哥哥竟然敢跟我吵闹，非要娶我的干女儿不可，这哪里是对养父的报恩？简直就是把我不当回事。

杨永寿一听果然是哥哥与毛泽东争女人而导致被送到朝鲜而死在了那里。毛泽东没放过我哥哥，他也不会放过我。想到自己大难临头，当即吓得小便失禁，旋即昏迷了过去。杨永寿在医院里醒来后觉得只有装疯卖傻一途了，便从此胡言乱语。当毛泽东得知杨永寿的确疯了，也就不再关心他了。杨永寿跟医生总是谈一些类似梦话毫无逻辑的言论，偶尔提到苏联。医生们想到他可是毛主席的儿子，也就提议去苏联给他治病。很快得到了上面的批复，杨永寿就再次去了苏联。本来他想在苏联度过余生，不用担心在毛泽东身边晃荡会丧命了，可中苏关系破裂，苏联便把毛泽东的儿子送回了中国。

然而，毛泽东还是不会放过这个曾经成功骗了他的乞丐，尤其对他是否真的得了精神病也是半信半疑，因为毛泽东对医生的话向来是存疑的。他担心这个孩子会继续骗他。被骗了的人总是神经兮兮的，一朝被蛇咬十年怕井绳，毛泽东也不例外。

杨永寿回到国内时要求到东北给他安排个地方，他不想回北京。这样，组织上也就答应了他的要求，让他在大连养病。

毛泽东得知他回国了，也没跟他打招呼就去了大连，便更加怀疑他在躲着自己。作为养子，越是有病越应该来到养父身边。可他总想离自己远点。他到底是真的有病还是装的，需要了解清楚。恰值寡妇张文秋又来看大女儿，便与毛泽东交谈起来。张文秋说起自己在北大读书的二女儿也到了该找婆家的年龄了，可惜岸青去了苏联。毛泽东告诉她那个傻孩子回国了，在大连养病呢，也不知道他的病有几分真几分假。张文秋便说，男人到了结婚年龄不结婚，没病也得闹出病来不是？给他娶个媳妇，说不定他的病就好了。毛泽东一听，当即眼前一亮：这可以看出来他到底是真病还是跟我装疯卖傻躲着我。俩人一合计，便给俩年轻人谋划好了婚姻大事。杨永寿得到毛泽东命令回北京与刘思齐的二妹结婚。

杨永寿可为难了！答应结婚吧，那个女人就是害死他哥哥的刘思齐的妹妹，这叫不是冤家不聚头；不答应吧？那就等于自己并不是真的又疯又傻，而是装的！如果暴露是在骗毛泽东而装疯卖傻，那可是死路一条。就只好装到底了。

杨永寿对女人早就在得知哥哥死了时彻底免疫了，他早已决定终生不近女色。可改名韶华的这个嫂子的二妹对这婚姻本来是抗拒的，因为她在北大已经有了对象---徐海东大将的儿子徐文伯。二人早就海誓山盟了。可她拧不过老妈的执着，对她的谆谆教诲便是与毛家攀上亲家是亲上加亲。韶华问母亲怎么会忍心把自己的女儿嫁给一个又疯又傻比自己大很多的男人。张文秋告诉她：只要一结婚，他的病就好了。他本来就是个很聪明的人，一旦好过来，就算嫁得值。

结婚后，杨永寿想起嫂子的妹妹就想起嫂子，那个毛泽东的干女儿。想起嫂子就想起了哥哥的惨死，便害怕得浑身打颤，哪里敢行肌肤之亲？不仅装疯卖傻，还在睡觉时躲着她。

当毛泽东得到韶华的汇报后便安排了一出留给后人研究其心理的一幕：毛泽东与韶华在杨永寿面前十指相扣，看看他的反应。毛泽东告诉韶华，这样可以刺激他一下，让他有反应。而内心里他是在测探杨永寿是否是装疯卖傻。如果是真的又疯又傻，那他就不会在意他媳妇与养父十指相扣暧昧无穷；如果他是装疯卖傻，那他的眼神表情变化便会显现出来。请看下面的两张图片。显然是同时照的，只是韶华的位置不同，但都是在丈夫杨永寿面前照的。杨永寿经得起考验，表情上没有愤怒或吃醋的显现，都是呆呆傻傻的样子。

韶华这招没令丈夫得到效果，便找到医生，要医生帮助解决丈夫无性生活能力的疾病，傻不傻倒先放在一边。医生无法得知杨永寿属于心理阳痿，因为他不知道杨永寿的内心世界里对这桩婚事是何等恐惧，便以为又疯又傻的他没性能力属于生理疾病。在那伟哥还没出世的年代，医生对此生理疾病爱莫能助。

没有不透风的墙，何况中南海内的故事都是在红墙内发生的。毛泽东与韶华十指相扣、与干女儿在丈夫死在朝鲜后又在中南海共同生活了八年才改嫁的事实，令中南海的人们背后议论纷纷，说毛泽东跟两个儿媳妇扒灰（参见李志绥的书）。杨永福死在朝鲜后新婚寡妇在中南海跟毛泽东又生活了八年的漫长岁月里，她每年夏天都跟毛泽东去北戴河。据罗瑞卿的女儿罗点点撰文说，刘思齐在北戴河只跟毛泽东有谈话的时候，其他人她理都不理。

显然，得知杨永福死了的消息后，刘思齐极大可能得了抑郁症。她变成了个谁都不理的孤独女人，有她自认为中南海人人知道她的秘密而无脸见人又不得不出来放放风而去北戴河的原因。

事实上这是对毛泽东的误解。且不说毛泽东是否与这姐俩有暧昧关系，就算有，杨永福杨永寿哥俩与毛泽东毫无血缘关系，算不上什么扒灰。在知情的毛泽东眼里，根本就没有扒灰的心理障碍。

然而，刘思齐韶华这姐俩、杨永福杨永寿这哥俩，加起来的四人，都是老天捉弄人的悲剧，都是值得同情的小人物。那些贬损她们姐俩贬损他们哥俩的人们，看到润涛阎这篇文章后就消停吧！以前的诋毁属于不知情的误解，可以原谅。

（十）杨永寿至死不承认他姓毛

杨永福杨永寿哥俩在强行被冯雪峰认同是毛岸英毛岸青哥俩后背送往苏联。在苏联以及从苏联回国到在朝鲜去世这段时间内，杨永福是否曾经以毛岸英的名义签署过信件，我没找到确切的证据，比如亲笔签名。有文章提到杨永福承认自己是毛岸英，比如他去参加土改时。但杨永寿则在任何时候都没有承认过自己姓毛。在苏联的时候，他的名字叫亚历山大.杨。在回国后他被派往东北参加土改，他以杨永寿的名字参加的。他走后当地人才传出他是毛岸青。从1960年在毛泽东张文秋二人的捏合下与韶华结婚后，他就回到了北京中南海家中隐藏了起来，谁也不见，根据已知材料，只是在1962年刘思齐带着改嫁后的新丈夫、韶华、韶华的三妹一起去见毛泽东时，杨永寿才一起跟着去见了毛泽东。在会见时他坐在与毛泽东最可能远的地方，目光朝向与毛泽东相反的地方照了合影。见图一。从此，他再也没见过毛泽东，包括毛泽东在弥留之际他都没去见一面这个养父。他低调生活着一直到毛泽东去世。毛泽东去世后他才公开露面。

1997年9月1日《大地》刊登的文章《在毛岸青、邵华家做客》中提到他去湖南板仓在签名薄上写下的是“杨岸青”。他很聪明，如果写杨永寿，大伙都不知道是怎么回事便会问他这名字的来龙去脉。但他不想留下自己曾承认姓毛的记录，便写下了“杨岸青”。好在杨开慧也姓杨，在场的大家就自然而然地认为他不想姓毛，喜欢随杨开慧的姓。可大伙还是以为他写错了。文章作者自以为是地认为是“他打开了幼年时与杨开慧感情的闸门！”

他的姓与杨开慧的姓碰在了一起显然是巧合。在湖南杨姓是大姓。

到目前为止，我们找不到杨永寿曾经亲自签名自己姓毛的记录。他不认同那个搞死他哥哥的毛泽东是什么伟人，而是杀死了他哥哥并拉着哥哥的老婆八年不改嫁，还当着他这个养子的面与韶华十指相扣，公然侮辱他。尽管他清楚是毛泽东在测试他是否装疯卖傻。所以，他也就不想认贼作父，坚持死不改姓。

（十一）是胡耀邦把机会留给了润涛阎

这里首先提出润涛阎第24定律：

政治骗子也是一步步从小骗到大骗走过来的。在这个过程中，骗子本人一定也经历过被骗。这就导致了一个恶性循环。所以，当你看到一个巨大的政治骗子骗人总是成功，有非凡人所能想得到的骗术，那表明他一定有过常人想象不到的被骗经历。拿毛泽东的例子来说，他的政治骗术达到了炉火纯青的地步，那一定表明他也有被杨永福杨永寿哥俩骗成毛岸英毛岸青的令人匪夷所思的令人难以想象的经历。这不仅仅导致被骗者毛泽东对假毛岸英毛岸青的摧残，也导致他对世人的不信任甚至报复。

说到这里一定有人会出来问：为何中央党校那么多研究毛泽东的教授、那么多研究政治研究心理学的专家教授、那么多毛泽东身边工作人员都没人发现这个上世纪里最大的欺骗事件？十亿人里为何只有你润涛阎独辟蹊径发现了这个天大的骗局？

其实答案很简单：整个地球上的人只有哥白尼一人破解了太阳不是围着地球转而是地球围着太阳转的谜底。如果说“真理只掌握在少数人手中”，那是指真理的总和，如果说某一个具体的真理，那往往只掌握在一个人手中。十亿人还是百亿人与发现真理一点关系都没有。那些研究毛泽东的专家教授们都是混饭吃的，他们如同我后院的蚂蚁，时时刻刻为了眼前利益而奋斗着。凡是与个人眼前利益无关的真理在他们和十亿人眼里一文不值，没有探讨的必要。而润涛阎是专门为了探索真理而来到人间的。

从理论上推理，胡耀邦应该是第一个发现这个上世纪最大骗局的。

毛泽东与他的所谓的儿子们儿媳妇们非同寻常的关系令胡耀邦内心里产生了对毛泽东人品的怀疑，不是所谓的路线之争权力之争那么简单。胡耀邦认为毛泽东是一个极端恶毒的暴君，比如他竟然把他早期的挚友、帮他把俩流浪儿子找到了的冯雪峰打成阶级敌人，残酷迫害自己的恩人。这就是典型的恩将仇报了。为了彻底清算毛泽东的罪行，胡耀邦需要找到冯雪峰帮毛泽东找到俩儿子的证人和来龙去脉，这样他才能在一大批毛左们面前说事实讲道理。可当他得知毛泽东是被骗了的受害者后，他自然也就不公开这个秘密了。这对中国那时候拨乱反正尽量肃清毛的流毒没好效果不说，绝大多数人是不相信有人胆敢欺骗毛泽东的。

按逻辑推理，润涛阎认为胡耀邦是明白了杨永福杨永寿哥俩不是毛泽东的儿子的第一人，因为胡耀邦看到了这些完整的证据链：

1. 经过胡耀邦派调查组的调查得知，毛岸龙由于在大同幼稚园传上了恶性痢疾而被送往医院，有医生证人，有他死在医院后幼稚园负责人买棺材给他下葬的证人。此时毛岸英毛岸青都在同一个幼稚园，幼稚园的领导人是地下共产党，他知道死的是毛泽东的小儿子，那他一定会通知毛岸英毛岸青哥俩，不会背着俩哥哥而暗自下葬，没这道理也没这必要。按照中国千年来的常理，毛岸龙下葬时即使幼稚园不是所有的孩子都参加葬礼，他俩亲哥哥一定会参加。而且在参加毛岸龙下葬时，毛岸英毛岸青哥俩会哭得死去活来。可找到的这个“毛岸英”竟然说毛岸龙是在他们哥仨乞讨时发高烧不能去乞讨（而不是在幼稚园时得的病），他带着毛岸青去乞讨，回来后毛岸龙就不见了。等于不是死了而是丢了。如果他真的是毛泽东的儿子毛岸英，那他一辈子也忘不了小弟弟死在医院，尤其是下葬时的场景。就这一条，胡耀邦就会判断出这个找到的死在朝鲜的“毛岸英”是冒牌货。

2. 真的毛岸英与毛岸青年龄只差13个月，而找到他俩的李云竟然说组织上告诉她要找的是两个湖南男孩，大的十三四岁，小的十一二岁。这句话表明她找到的俩人相差不是一岁的样子，她才篡改组织上让她找的俩孩子的年龄差别。组织上非常清楚这俩孩子只差一岁，应该告诉她：“两个十三四岁的孩子”才合乎情理。中央特科花费了几年的功夫和大量的金钱找人，连找的俩人差多少岁都不清楚，那共产党特科里的人也太蠢了，根本就不适合干这行。胡耀邦看到李云的交代，便知里边的猫腻了。何况胡耀邦在延安时也说不定亲眼见过找到的这哥俩年龄上相差即使不超过三岁也绝不是一岁就纳闷过呢，这次当上了党主席他才有机会下令对此事调查个清清楚楚明明白白。

3. 毛泽东对帮他找到两个亲儿子的冯雪峰恩将仇报。就算毛泽东是个无情无义的政客、暴君，冯雪峰对他没任何威胁，根本就不是他的政敌，他有何必要对恩人如此绝情？

4. 为何找到这俩孩子的李云对此事件隐藏得死死的直到毛泽东去世才公开？这是天大的功劳，她为何不说出真相？尤其是当她听到毛岸英死在了朝鲜成了烈士后，她应该邀功才对。可当她知道毛泽东在那你方唱罢我登场的内斗中真的战胜了张国焘王明等对手而获得了天下后，她后怕死了。好在冯雪峰独吞了这功劳，主要是因为冯雪峰是毛泽东的挚友，而毛泽东根本就不认识李云一个干活的，加上冯雪峰人品还说得过去没在挨整时把李云咬出来，她才在战战兢兢的日子里逃过了毛泽东血淋淋的铡刀。她清楚毛泽东为何要把自己的长子送往朝鲜而很快就死了的原因，全国人民此时就是她清楚这里边的猫腻。胡耀邦想到这里，就明白了为何毛泽东不让把毛岸英的尸体或骨灰运回国埋葬在杨开慧墓旁。杨开慧死时最担心最挂念的是三个孩子。长子死了，把尸体葬在妈妈身边，是最起码的对杨开慧尤其是对毛岸英的尊重。他在九泉之下也想回到妈妈身边。恰恰是因为他不是杨开慧所生，毛泽东才不能让他的尸体埋葬在杨开慧身边，这是毛泽东对杨开慧负责的态度。胡耀邦此时对毛泽东如此处理这件事理解了。

5. 中南海都知道毛泽东与两个儿媳妇关系不正常，一个是丈夫走了不告诉她去哪里，三年不见面没收到信，新娘子竟然不闻不问，而且公公就在中南海的家里。这不符合常识。毛泽东还与韶华十指相扣照相，还是在毛岸青在场时。这是公开对儿子的侮辱。毛泽东有必要当面羞辱自己的亲骨肉而且他还是有病之人？就是看在杨开慧的面子上，再缺德的人也没必要这么对待自己的儿子吧？只有当胡耀邦知道这俩女人与毛泽东根本就不存在儿媳妇的关系，他也就理解了毛泽东与韶华十指相扣是测试假儿子是否真的又疯又傻还是装的。

6. 文革开始后，江青在北大公开讲话时突然说：“阶级斗争搞到我家里来了。特务搞到我家里来了。张少华（韶华的名字）在不在？再看看张承先的干部路线，在领导核心中有个张少华。她的母亲张文秋是全国通报的政治骗子，张少华她自己说是毛主席的儿媳妇，我们根本不承认。”她边说边哭。

只有当胡耀邦清楚了毛泽东与这找到的俩假儿子的关系，他也就理解了江青的所作所为。

7. 打从1962年，毛泽东再也不见“傻儿子”。按照常理，自己的孩子越是有残疾，当爸爸的越是心疼，越是倍加呵护。毛泽东在临死前躺在床上时，如果这个傻儿子真的是他亲生的，那他一定会把他叫到身边。人之将死其言也善，这是最普通的道理与常识。可毛泽东就是快死了也不见一见他唯一活着的在中南海身边的儿子。很多在海外的儿子得知老父处于弥留之际也长途跋涉往回跑去互相看最后一眼。况且在毛泽东死后，毛岸青出来露面，并没有精神分裂症患者的举动，他理应在父亲弥留之际去见一面，可他就是不去。毛泽东还有个六岁的孙子毛新宇，也在中南海，可毛泽东就是没见过他一面。只有当胡耀邦清楚了这个毛岸青根本不是毛泽东的儿子，才恍然大悟，一切过去无法解释得通的现象一下子就都解释得通了。

8. 这个傻儿子至死不说自己姓毛。在1979年他被邀请去湖南板仓参加纪念馆落成仪式，他在签到簿上在众目睽睽之下落款是“杨岸青”。这也是胡耀邦起初纳闷的地方。当他清楚了他本来就不姓毛，而且他装傻充愣才活了下来，胡耀邦认为他是个聪明人，而且一生与哥哥的小姨子不行男女之事。至于毛泽东如何对待这俩女人，与他杨永寿毫无关系。“爱怎么折腾就怎么折腾吧。我失去了哥哥换了个养父，远不如跟哥哥一起讨饭度日时更有尊严更自由，所以，我不会姓你那个毛。”

所以，润涛阎猜测，从以上八条胡耀邦理应得出杨永福不是毛岸英，杨永寿不是毛岸青的结论。而且整个乌龙事件的来龙去脉非常清楚。

那为何胡耀邦不把真相留给他人以便未来曝光？

这有两种可能：一是胡耀邦也是对真理视而不见之人；二是他清楚这样的弥天大谎世纪之最大骗局一定会有润涛阎的文章横空出世。

（十二）看图说话（证据在文章里，图片也有点附加值）

我回国时在机场吓了一跳。接我的人中还有个人，他是我大舅。可我大舅早就去世了，他怎么会来机场？原来他是我大舅的三儿子。可我怎么也对不上。我最后见到这个表弟时他还是个孩子，长得一点不像我大舅。因为我没见过我大舅小时候是什么模样。

有人说粟裕当时在准备打台湾。可事实是，当时的中国百废待兴，后勤和经济上武器弹药等根本没能力同时打两场战争。毛泽东必须在要么抗美援朝要么打台湾两者之间最多选一个。有人说粟裕身体不好。他那个弹片是多年的老毛病了，在淮海战役时他也头痛。抗美援朝战役中和以后多年，他那个弹片也没取出来，他照样当总参谋长竟然与彭德怀聂荣臻等争总参的权力范围呢。

毛泽东派去跟彭德怀一起去朝鲜的助手没有一个是死了值得毛泽东惋惜的，十大将军里一个都没去。不是毛泽东不想让他们去，而是等彭德怀和假太子等志司的人被炸死后林彪挂帅时再去。彭德怀的得力助手大将黄克诚都没跟着彭总去朝鲜。邓华、洪学智二位如果没有抗美援朝当上副总司令，在1955年评军衔时能否得到上将军衔都有疑问。

毛泽东为何要彭德怀带着太子在身边给彭总招致更多麻烦？难道中国就找不到一个可靠的俄文翻译？到关键时刻彭总是顾打仗还是顾保护太子？要论对太子的保护能力，不论邓华还是洪学智都不在这方面有特别的经验。毛泽东最看重在这方面最有能力的是罗瑞卿和陈赓。毛泽东一打下天下就把高官们的安全责任交给了罗瑞卿，因为他信得过罗瑞卿在这方面的能力。陈赓当过蒋介石的警卫，在共产党里也干过特科，相比之下，邓华洪学智跟他们比在这方面是没资格保护太子的。如果杨永福是真太子，毛泽东无论如何也会派罗瑞卿或陈赓去朝鲜，何况论打仗的资历罗瑞卿陈赓要比邓华洪学智强多了。

（十八）毛泽东为何不作诗填词怀念长子爱国民族英雄毛岸英？

从毛泽东对他侄子的呵护提拔栽培便可得知如果死在朝鲜的杨永福是真实的毛岸英，毛泽东对他的死一定苦不堪言。这也解释了他对俩冒名顶替的骗子成了他的养子后出手狠辣：一个被烧死，一个生不如死靠装疯卖傻活下来还要遭受亲眼见到公公与儿媳妇十指相扣的羞辱。即使如此都难发泄内心里对他们仇恨的十之一二。

毛泽东喜欢作诗填词，就是无所事事的时候还给一位女工作人员参加民兵打靶训练照片作诗一首《七绝-为女民兵题照》。杨开慧死于1930年，而毛泽东与贺子珍在1928年结婚。毛泽东没资格表达对杨开慧有牢不可破的真爱，可他还是填词一首《蝶恋花》。而他的长子皇太子毛岸英死在了朝鲜，那可是民族英雄，毛泽东本应该作诗填词怀念这亲骨肉，也能激发全国人民的爱国热情。然而，毛泽东即使给女民兵作诗七绝，给被他抛弃的前妻填词蝶恋花，他也不会给自己的亲儿子爱国民族英雄毛岸英作一首七绝填一首蝶恋花。因为他不是真实的毛岸英，他是杨永福。是狸猫不是太子。

（十九）逻辑推理与亲子鉴定

真理的发现者是依靠强大的慎密的逻辑思维能力在芸芸众生视而不见的事件中借助渊博的知识、天马行空的想象力把真相从迷雾中找出来。而芸芸众生则是靠感官看表面现象来给予描述、解释。如果猴子也有语言能力，我们告诉猴子地球是个球体，太阳也是个球体，太阳从东方出来从西方落下，问猴子们是地球围着太阳转还是太阳在围着地球转，猴子们便能得出太阳围着地球转的结论。这就是芸芸众生无法发现真理的缘故。当真理的发现者通过强大的逻辑思维能力准确判断出真相的时候，芸芸众生的第一个反应不是去研究这些逻辑是否有破绽，而是先想到眼睛看到的比如照片。从对真理的第一反应来看，芸芸众生的智商跟猴子差不多。所以，越是惊天动地的真理被发现，后面急得跳脚的猴子就越多。

哥白尼用强大的逻辑思维推理出地球围绕太阳转后，布鲁诺醍醐灌顶，然而，在他被烧死之前无数人认为只有他能到天上去看看到底是太阳围着地球转还是地球围绕太阳转才行，否则，哥白尼就是胡说八道。后来的小人物们用天文望远镜甚至通过卫星彻底证明地球围绕太阳转，这些“证明”是为了说服芸芸众生的，不是为了说服哥白尼的，哪怕哥白尼还活着。因为哥白尼心里跟明镜似的。门捷列夫根据已有资料通过强大的逻辑推理能力得出元素周期表这个真理，那些小人物们会去寻找还没发现的各号元素，最终以“证明”门捷列夫周期表是真理。这些“证明”是为了说服芸芸众生的，不是为了说服门捷列夫的，因为门捷列夫心里跟明镜似的。物质含有的能量有多少？不同的物质含有的能量是否一样多？爱因斯坦根据严密的逻辑推理得出：物质的能量等于该物质的质量乘以光速的平方。那些小人物们便去想方设法打开原子核，看看原子核打开后是否能放出如此巨大的能量。这类“证明”是为了说服芸芸众生的，不是为了说服爱因斯坦的，因为爱因斯坦的心里跟明镜似的。今年最大的科技新闻莫过于测到了引力波。那些小人物们终于“证明”了爱因斯坦提出的引力波的存在。其实这类“证明”是为了说服芸芸众生的，不是为了说服爱因斯坦的，因为爱因斯坦通过慎密的逻辑推理得出的相对论是真理，爱因斯坦心里跟明镜似的，不需要他人证明。上面说的小人物们是指在用逻辑推理得出真理的伟大人物们眼里是小人物（这要看跟谁比），而在芸芸众生眼里他们就是伟大的科学家了。所以，爱因斯坦说过：“有两个东西是无限的，一个是这个宇宙本身，一个是人的愚蠢。对于宇宙是否真的无限，我不敢肯定。”

润涛阎根据已知资料用慎密的逻辑推理得出杨永福杨永寿不是毛岸英毛岸青，芸芸众生们当即去找他俩的照片，而非享受润涛阎无懈可击的逻辑推理过程。就好比你问猴子是地球围绕太阳转还是太阳围绕地球转，猴子们通过太阳从东方升起从西方落下便得出太阳围绕地球转，靠的是感官，而非大脑思考。

有网人提出：你润涛阎根据毛泽东在儿子面前与儿媳妇十指相扣不合情理，那是因为你用常人的行为推理毛泽东，而毛泽东不是常人，他就用与亲人十指相扣表达亲情。

这说法没问题，润涛阎绝对认同。但你必须清楚用同样的逻辑标准思考：毛泽东对侄子毛远新的关怀与培养、提拔远远超过对他“儿子”毛岸青。为何毛泽东不以毛泽东才具有的亲情表达方式在毛远新面前与毛远新的媳妇十指相扣？要知道，毛泽东与韶华十指相扣的数张照片都是韶华留下来并公开出来的，是在电脑PS技术发明出来之前，所以绝对当真。

有网人说：毛泽东是伟人，不把儿女情长放在眼里，心里时刻考虑的是人民大众。所以，他在临死前也不把傻儿子叫到身边顺便把孙子带来让唯一的孙子（也住在中南海毛泽东的身边）看一眼爷爷。

这说法润涛阎绝对认同。但你必须清楚：如果你这逻辑合理，那毛泽东可是在临死前把远在东北的侄子叫到身边。侄子为何要比儿子更亲？早在杨永寿从苏联回国时，杨永寿就在东北搞土改了，而且能用熟练的中文舞文弄墨，在土改时纠正了当地的错误做法而得到了当地干部群众的一致好评。如果他是真的毛岸青，有才有德，本应受到毛泽东的栽培才对，那他为何回到中南海毛泽东身边就傻了？在苏联好好的，回国后搞土改工作生活都好好的，还能舞文弄墨呢。官方给出的解释说是他在小时候讨饭时被打坏了大脑，得的是精神分裂症。那为何他在苏联近十年都没犯病？回国后还能搞土改，还能纠正当地领导的错误，还能舞文弄墨，为何到了中南海毛泽东身边他突然就犯病了？而等毛泽东一死他就又出头露面了？而且从未有过精神分裂症患者的行为，他还接受采访，还到湖南参加活动，并签名呢。这样的身体精神状况，毛泽东为何死前不见他一面跟他告别人生？反而跟侄子天天在一起？

如果找到的会说湖南话的俩乞丐杨永福杨永寿真的是毛岸英毛岸青哥俩，那为何找到他俩的李云一生都不敢出来邀功？只是到毛泽东死后才说出是她找到俩人的？董健吾帮助毛泽东照顾、用性命保护并找到了毛岸英毛岸青哥俩还把他们送到苏联，对毛泽东来说功勋卓著，那为何他不跟毛泽东邀功？反而一生中低调低调再低调？他本是周恩来的下属，为何他低调一生而周恩来却不提拔重用他这个毛泽东的恩人？就是看在毛泽东的面子上，周恩来也会大力提拔重用他并向毛泽东介绍是他用生命保护了毛岸英毛岸青哥俩。

润涛阎告诉你：因为精明的周恩来从董健吾如此低调这一点就可清楚那俩孩子是冒名顶替的。

如果周恩来认为死在朝鲜的毛岸英是真的，周恩来就会认为毛泽东不让把太子的尸体或骨灰送回国是毛泽东知道死在朝鲜的其他烈士不把尸体弄回，他就不能带头，那口若悬河的周恩来就会找到借口，说是全体志愿军们的呼声要求毛岸英葬在杨开慧墓旁以说服毛泽东。甚至他根本就不需要说服毛泽东，来个先斩后奏。别说周恩来不会干涉毛泽东家庭的私事。据黄永胜的儿子写文章披露，周恩来在毛泽东不知情的情况下让医生给韶华人工授精才生下了毛新宇。事后周恩来与黄永胜去到毛泽东那里邀功，毛泽东毫不感恩并冷冷地说了一句：又多了一张吃饭的嘴。

这就延伸出一个逻辑问题：周恩来清楚那俩孩子不是毛泽东的儿子，那为何周恩来还要通过人工授精给毛泽东留下个假孙子在人间？

毛泽东到了晚年一直都在想周恩来是否在第一时间就知道杨永福杨永寿不是真的毛岸英毛岸青。虽然冯雪峰不是周恩来的人而是毛泽东的挚友，可他那真实的俩儿子毕竟是在上海特科人的手中活不见人死不见尸的，那时候冯雪峰还没去上海呢。毛泽东不会把找到俩假货的责任放在周恩来头上，但毛泽东的俩儿子在周恩来的特科人手里没了，周恩来是否知情？凭周恩来的精明，他早在延安时就会通过康生等人转告毛泽东他周恩来给在上海特科保护毛岸英毛岸青的人每月三十块钱，以表示周恩来对毛泽东无限忠诚。突然间俩儿子生死不明，周恩来真的就不知道是在谁的手上搞丢的？两年后突然间又找到了，特科的人就没人汇报给周恩来？对这些疑问，毛泽东的脑子里是无法消除的。

即使毛泽东后来清楚他俩儿子早就被人出卖了，可在那兵荒马乱的战争年代，连他自己的性命都不知道何时丢掉，查出来到底是谁出卖了他俩儿子，已经没有意义了。更重要的还是上面提到的原因：毛泽东不想让人民大众和他的政治对手们各国反动派们知道他断子绝孙，而是设法把不利因素变成有利因素而为自己所用。然而，周恩来必须表现出他与此事无关，表演不能不到位，更不能过头。他帮韶华人工授精给毛泽东弄出个孙子，在毛泽东那里丝毫不领情，表明毛泽东是在猜测他周恩来干这事是在戏弄他还是在巴结他。毛泽东对医生历来半信半疑，他对这个孙子的精子到底是不是那个傻子的还会存疑。反正都与毛家无关，他也就不关心这人工授精出来的孩子了，死都不会看他一眼。

如果周恩来确定毛新宇是毛泽东的亲孙子，周恩来一定安排让毛泽东在中南海溜达时“无意间”碰上这个孙子。周恩来通知一下汪东兴就办到了，易如反掌。毛泽东见到自己唯一的真孙子应该是非常高兴的，毕竟他也是有血有肉的人。老虎都有舔犊之情，何况最终没有“断子灭孙（毛泽东原话）”的老人。就这一条就把当年在庐山会议上三次提到自己无后的“阴影”变成了“明灯”：毛家后继有人，毛泽东的事业千秋万世代代传。

有人说毛泽东是神，你润涛阎不能用人的行为看待毛泽东。

对此，润涛阎绝对认同。但你别忘了，神是万能的，是不需要吃饭就不会饿的。你什么时候听毛泽东身边的人说过毛泽东三年不吃不喝也不饿？报道的是毛泽东喜欢吃红烧肉等后来的“毛家菜”，没听说过他不吃饭也不饿。那表明他还是一个有血有肉的人，而非什么神。耶稣是神，可耶稣的妈妈是处女就自动怀孕生了耶稣；刘邦是神人，介于神与人之间的玩意，可刘邦的妈妈不是跟刘邦的爸爸做爱怀孕生的刘邦，而是一条蛇爬到他妈妈身上跟他妈妈交配了怀孕生的他。这是刘邦自己说出来的。毛泽东从来都承认他是他爸爸毛顺生与他妈妈的亲骨肉。所以，毛泽东不属于不吃饭不饿的神也不属于类似刘邦介乎神与人之间的老妈跟蛇交配出来的神人。

杨开慧死后，决定把毛泽东和杨开慧生的三个儿子交给上海大同幼稚园的是毛泽东的弟弟毛泽民。毛泽民也是在上海的地下党，主要工作是印刷共产党需要的文件。但他在1927年就把印刷厂搬到了天津，因为他知道上海是戴笠的大本营，危险太大，而戴笠在天津的人手少得多，就安全一些。理论上推理，他是得知嫂子被杀三个侄子毛岸英毛岸青毛岸龙在湖南老家太危险而决定把他们转移到上海的。

那毛泽民为何在自己逃离戴笠的虎口转到天津后却把三个侄子送入虎口？他为何不把他们仨安排在天津自己可以照顾一下？比如在天津找到朋友把三个侄子隐藏起来？如果钱不是问题，那在天津找到熟人以领养的名义隐藏总比在戴笠大本营的上海安全些。否则他自己何必搬到天津去搞印刷厂？至少应该把风险分散一下，比如带一个到天津。把三个孩子都放在一个地方，而且是非常危险的地方，不符合常识。这事不能怪湖南地下党组织，也不能怪董健吾。仨孩子是毛泽民的亲侄子，毛泽民有权力给他们做出安排，外人无权干涉。董健吾总不能拒绝保护毛泽东的孩子们，他最多把风险告诉毛泽民。剩下的只能是看命运的安排了。

其实最让我不可思议的是：在顾顺章叛变后，知道毛岸英兄弟在上海地下党的幼稚园一定有巨大风险，毛泽民理应当即扑到上海找到三个侄子，把他们带出虎口。要知道，此时毛泽民在苏区根据地，是那时共产党最安全的地方。即使不把他们带到苏区，去任何地方也不能让他们在上海，因为共产党上海特科人员都在撤离。可毛岸英兄弟在大同幼稚园关闭后的一年半时间里，他俩都在董健吾家中，毛泽民在这一年半的时间里就没去找他们。在上海地下党工作多年的毛泽民是非常容易找到董健吾的。这实在令人费解。由于毛泽民死前未留下他这么做的只言片语，我们无法猜测其真实原因。从顾顺章被捕到毛岸英毛岸青离开董健吾家这一年半多的期间，毛泽民正在苏区当中华苏维埃政府的银行行长。难道工作要比侄子们的性命还重要？

如果大胆给出解释的话，那唯一可以解释的原因便是毛泽民认识到去上海接孩子太危险，为了俩侄子（事实上是三个，因为那时他还不知道毛岸龙已死）的性命把毛泽民的命也搭上不值得？那毛泽民也应该安排一下找一个可靠的人去把他们仨接出来才合情理。总不能说为了革命而把无辜侄子们的命视如草芥吧？如果毛泽民坚持要去上海接回侄子，毛泽东能阻挡得住他？甚或贺子珍不让毛泽民去接回太子们？没有资料可分析，内情我们便无法得知。

所以，是毛泽民对侄子们的性命大大咧咧而导致毛岸英毛岸青活不见人死不见尸。他是最大的责任人无疑，哪怕后来李云找到的是真太子，他对处在虎口的侄子们不去搭救，实在令人费解。

（二十五）毛泽东是怎样给假太子与彭德怀布局的？

上篇提到，毛泽东出兵朝鲜的目的最关键的三点是借美国之刀杀人：炸死欺骗他的假太子、炸死“跟毛泽东三分合作七分不合作（毛泽东原话）”的彭德怀、杀掉国民党降卒以给林彪四野瘦身。所以，不能让彭德怀和假太子得知真相，必须把此时的他俩以及后来的全国人民埋在鼓里，是毛泽东必须考虑的。毛泽东需要巧妙布局，要给彭德怀与假太子灌足迷魂汤。

1. 临走前给杨永福（假太子）面授机宜

杨永福是在头天晚上才得知去朝鲜的。到底毛泽东与假太子在临别前的晚上说了什么，目前没有任何文件披露，好在这是明摆着的最简单的布局，政治家肚子里的蛔虫润涛阎可以还原当时毛泽东是怎样告诉假太子的。毛泽东的原话虽然可能与润涛阎所说有些许出入，但内容绝对不会差得离谱。

为了减除假太子不想在新婚之际去战场而且是跟美国为首的联合国军对垒所存在的天然恐惧感，毛泽东这样告诉假太子：

“岸英，你是毛泽东的儿子，将来要接我的班当主席的，你需要有接班的资格，这次抗美援朝就是一个机会。你不用害怕，我安排好了，你只要按我说的去做就好了。”

“爸爸，我听您的。您怎么说我就绝对照办。”

“那就好。第一，你不能离开志愿军司令部（志司），要时刻跟彭德怀在一起。”

“您是说，我不能去下面锻炼并指挥战斗？”

“不能。你不能冒险。你离开彭德怀一步，说不定他就背后杀了你，断绝我把江山交给你的路。”

“那好。”

“第二，在任何情况下都要坚持志司的电台在你身边，如果任何人包括彭德怀提出电台要放在离志司远的地方，你就提出‘这不行，因为我需要随时跟我爸联系，说不定他随时都会有新的指示，包括苏联斯大林的指示。电台必须在我们身边。’这样，我随时都能给志司包括彭德怀下命令。彭德怀是一个刚愎自用之人，他并没有指挥过如此大兵团作战战役，需要我给他下命令。可他自视甚高，以电台不在身边为由不听我的话而自作主张。”

“这一点我绝对能做到。我认为我只要把道理讲清楚，大家都会赞同。”

“第三，你虽然身份是俄文翻译，事实上我不会让斯大林直接跟彭德怀联系的，你并没有翻译的任务，可你没有资格当总司令的参谋，我不得不给你想出个办法让你去镀金。然而，在彭德怀与其他副司令等人讨论作战计划时你要争先恐后地发言，这样，未来就会有人把战争功劳记在你的头上。你需要大胆否定彭德怀的建议，打仗这事我最清楚，总有意想不到的战役失误。到时这些失误就能算在彭德怀的头上，而你当初是反对彭德怀的主张的。”

“我明白。一定照办。”

“第四，你必须找一个机会骂彭德怀的娘。”

“骂娘？就是三字国骂？”

“不论是什么，必须是带有‘娘’或‘妈’字。”

“那其他人听到了怎么办？”

“不怕任何人听到。彭德怀这人我最了解，你要是不骂他一次，你就别想镀金占他便宜。你越是骂他娘，他越是尊重你。记住，就骂一次。否则，他说不定鱼死网破杀了你。骂了他后，还继续跟他搞好关系。”

“就是说，我在他讲话时插言，令他不高兴指责我，我就骂他娘。那如果不论我怎么指手画脚他也尊重我呢？”

“那就在生活方面找机会。反正你必须找机会骂他一次娘。”

“行，我能做到。比如跟他下棋。就找机会骂他一次娘。”

“那更好。”

“既然斯大林不会跟彭德怀直接联系，那要我这个翻译有点不合适，为何不能给我一个参谋的头衔？”

“彭德怀是什么人？我儿子要是个翻译，他还能容忍，甚至保护你。如果我让我儿子给他当参谋，他认为你是去镀金的，是占他便宜的，他会不客气地跟你发怒。你反而无法在他身边工作了。”

“明白了。我一定遵守您这四条指示。”

“另外，你去朝鲜这事不能告诉松林，以后还是由我慢慢做她的思想工作。明白吗？你今天晚上就在这休息，准备明天出发。”

“好的。拖您做松林的思想工作了。估计她能理解爸爸的安排，是为我的未来着想，这样一说她就理解了。”

“你到朝鲜后也别给她写信，因为你的身份是保密的，只有彭德怀知道。志愿军司令部发出的信极有可能会被敌人截获，那你的身份就曝光了，你的生命危险就很大。”

“好的，我不会往国内发一封信的。”

“休息吧，明天一早就上路呢。”

当志愿军司令部（志司）听从彭德怀对战局的分析时，没等副司令和参谋们议论，假太子就站起来指指点点，令在场的将领们目瞪口呆，直等着彭总呵斥这不知天高地厚的年轻翻译。可彭总一言不发。这就给假太子遵照毛泽东“找机会骂彭德怀的娘”带来了麻烦，总不能在彭总不训斥他时就骂人吧？所以，等到一天他跟彭总下棋，彭总悔棋时，假太子当即骂娘：“你他妈的又悔棋！”只骂得在一旁的副总司令惊呆成傻子状。彭总还是忍下了。彭总的脑海里出现了这样一幕：在自己出来打天下并加入共产党以来，唯一被骂娘的是在延安时，那是在擅自打百团大战后被毛泽东骂娘四十天。今天，又被骂娘，当真是有其父必有其子啊。从此，志司的任何人问起彭总这么没教养的翻译当真是毛泽东的儿子？长得也没毛泽东的影子啊。彭总当即发誓：这要是错了，我彭德怀不是人！

从此，彭德怀再也不会怀疑这太子是假的了。真太子在身边，他就不会怀疑毛泽东不停地跟他电报联系的目的是招引美国侦察机。至于是否发生过志司有人提议把发报机放到另一处而遭到假太子的反对，没有看到任何资料披露。也许刚愎自用的彭总就认定了是金日成一直在泄密而导致志司转移到哪里轰炸机就跟到哪里，与电台频繁发报无关。

2. 毛泽东告诉彭德怀选他去朝鲜的理由

根据逻辑推理，毛泽东既然派林彪的四野为主力去朝鲜，那理应由有指挥大兵团作战经历的林彪亲自指挥才对。可毛泽东此时想杀掉的是彭德怀。如果让彭德怀而非林彪去指挥，那得找到合理的理由。

这事难不倒大政治家毛泽东。

他先找林彪谈话。就说苏联斯大林要我们出兵朝鲜，现在百业待兴，我还没定下来去还是不去。林彪是什么人？“读毛主席的书，听毛主席的话，照毛主席的指示办事，做毛主席的好战士”是林彪的座右铭。他从毛泽东的口气里便得知毛泽东不想出兵，当即讲：主席您开个会，在会上我提出不出兵的好处。这样，要是斯大林怪罪起来就说手下的将领反对出兵。

这就是毛泽东设计出来的“林彪反对出兵朝鲜”的一幕。表演这一幕时，毛泽东派专机去大西北接彭德怀回京。周恩来告诉彭德怀：政治局会议来不及等你就开始了。

林彪这一幕表演完了，彭德怀才进来。当彭德怀知道了两个观点后，觉得还是要支持一下出兵朝鲜一派。根据资料披露：

“第二天上午，毛泽东约彭德怀单独面谈，告诉他中央书记处关于出兵援朝的意见，同时说明原来准备林彪率军入朝，但林彪不赞成这项决策，加上过分看重自己的常胜经历，害怕战败，藉口有病不能受命。因此，‘我们的意见，这担子还得你来挑，你思想上没有这个准备吧？’彭德怀爽快地回答：‘要说思想准备么？确实一点也没有，但我服从中央的决定。我还不服老呦。’”

“10月5日下午，中共中央政治局再次开会讨论是否出兵援朝问题，仍有两种截然相反的主张。彭德怀在发言中明确指出：出兵援朝是必要的，打输了，最多等于解放战争胜利晚胜利几年。可是，如果让美军摆在鸭绿江边和台湾，它要发动侵略战争，随时都可以找到借口。如果让美国占领了朝鲜半岛，将来的问题更复杂，所以迟打不如早打。听了彭德怀的慷慨陈词，周恩来露出了满意的笑容。毛泽东最后宣布：我和恩来、少奇、朱老总等同志商量了一下，想让彭德怀同志率兵出征。我们也和彭德怀同志说了，他慨然允诺。好吧，德怀同志，我感谢你，中国人民感谢你，你是临危受命啊！”

会后，毛泽东再找林彪谈，说彭德怀单独跟我谈时他坚决站在出兵一方。恩来少奇朱老总也都支持他带兵去朝鲜。我想先让他去带兵，如果他玩不转，你立刻上场。我看他彭德怀没指挥大兵团作战的经历与能力，可不让他去他就不服气。等他败了，你再去，不仅他彭德怀，其他所有人都得服你。

林彪也就等着亲自赴朝鲜指挥的消息了。

那彭德怀走前毛泽东应该怎么跟他交代呢？

第一，先重复一下事情的来龙去脉：林彪公开讲他不想去。可我觉得林彪都不敢去，其他人就更难办了。当然，如果我要让有指挥大兵团作战经历的粟裕去，他也得去。可那不是主动的，打仗这事，主帅不能被动。我想到了你老彭。我说过，论横刀立马，唯有彭大将军。当然，如果你也认为去朝鲜跟美国对打没胜算，那我们干脆就不出兵了（毛泽东对彭德怀想当民族大英雄的个性了如指掌，彭德怀如何答复他，他早就知道了，比彭德怀知道的还早呢）。

第二，我派你去也有另一方面的考虑。其他人带兵去朝鲜，他们从内心里必然认为金日成是跟我毛泽东一个级别的国家领袖，最多差那么一点点而已，他们就会听从金日成的调遣。那最后结局等于：败了，是中国志愿军的责任；胜了，功劳是金日成的。我们白死人，是给人家干活的。如果你去，那就不同了，你完全可以彻底撇开金日成，骨子里都不把他当成主人，你就是志愿军总指挥，仗怎么打，不跟他金日成商量。你也可以不跟我商量，你在朝鲜期间你就是主席，你回来后我是主席。

# 上世纪最大骗局：毛岸英毛岸青是假的（续三） (2016-05-09 20:25:38) 下一个
（二十六）在继续写下面的故事前需要对一个关键事件进行探讨。那就是：真实的毛岸英毛岸青哥俩是怎么丢失的？是在谁的手里丢失的？是什么时间丢失的？

（1）毛岸英毛岸青在黄慧光家里的时间

根据董健吾的女儿董霞飞在《特科奇人董健吾》中的说法，原文照登：“岸英兄弟在黄慧光家大约住了四个年头。”

根据资料，毛岸英毛岸青哥俩在董健吾家一直待到1933年8月份才去了黄慧光家。毛岸英毛岸青被找到后送往苏联的时间是1936年6月。按照董霞飞的说法，毛岸英毛岸青哥俩在黄慧光家待了大约四个年头计算，1933年8月加上4年就是1937年8月。也就是说，当被找到的毛岸英毛岸青送往苏联一年零二个月后，真实的毛岸英毛岸青还在黄慧光家呢！因为里边说的是大约，那就把2个月去掉，或者干脆减少8个月（根据四舍五入的原则，如果低于半年取整数的话就应该说是大约三年，故在此减去8个月），也就是说，被找到的假的毛岸英毛岸青哥俩去了苏联半年后真的毛岸英毛岸青还在黄慧光家呢！二人从未流浪过。

这难怪当刘思齐演讲时重复丈夫杨永福诉说自己当年跟讨饭度日的三毛一样流浪了5年（刘松林早期的回忆说丈夫告诉她的是流浪6年）导致董健吾的后人状告刘思齐。

也就是说，根据董霞飞的说法，真实的毛岸英毛岸青哥俩从1931年初到上海到1937年8月左右从未过过哥哥带着弟弟乞讨流浪的“三毛”生活。

根据董健吾后人董龙飞状告刘思齐的诉状来看，对毛岸英毛岸青哥俩在黄慧光家的时间是不肯定的。依照法律规则，如果毛岸英哥俩在黄慧光家的时间是大约4年（超过3年半才能说成大约4年，而非大约3年），那在状纸上就应该准确无误地表明是超过3年。可状纸上写的是：“兄弟俩从1933年8月来到黄慧光家的，1936年6月党组织送他们前往苏联，就算毛岸英兄弟俩刚到黄慧光家就立刻出走，流浪时间最长也不会超过2年10个月。”里边没有给出哥俩出走的确切时间。

根据李云的说法是，1935年特科开始寻找乞讨流浪的毛岸英毛岸青哥俩。这跟董霞飞说的对不上。根据李云的说法，毛岸英哥俩在黄慧光家住到不会超过1935年中旬，也就是说在黄慧光家不到两年就出走了。这跟董霞飞的说法差了两年。

小结：

董霞飞的文章不可靠。毛岸英哥俩在黄慧光家住大约4年如果当真，别说俩假的都去苏联了他们还在黄慧光家住着呢！那党组织为何派人去寻找他俩？董霞飞的说法也印证了杨永福告诉妻子刘松林他带着弟弟跟三毛一样流浪6年（或5年）的经历与真实的毛岸英哥俩完全不符。表明杨永福杨永寿与毛岸英毛岸青在经历上风马牛不相及。

（2）真实的毛岸英毛岸青在何地丢失的？

根据官方的说法：“党组织每月给30块钱。后来，党组织没钱了，董健吾失去了固定收入，黄慧光是家庭妇女没有生活来源，自己有四个小孩，加上岸英兄弟一家七张嘴天天要吃要喝，日子过得艰苦了起来。生活艰苦，吃不饱肚子，还要挨打受骂。岸英自小就爱憎分明，一天，他看见岸青又挨了打，便离家出走了。”

也就是说，毛岸英哥俩被董健吾的前妻黄慧光收留并被照顾，是因为党组织给她钱。这是合乎情理的。虽然离婚了（在那年代离婚双方往往是仇人），但看在钱上，这等于是生意。

如果我们把黄慧光、毛岸英哥俩看成是人而非神，那么我们可以做出以下简单的推理证明：

董健吾把黄慧光当成生意人“我出钱你照看俩孩子”等于把黄慧光当成毛岸英兄弟的保姆（在上海那时候怎么称呼此身份不重要，内容就是保姆或幼儿园阿姨）。当黄慧光得不到看孩子的血汗钱（包括提供给俩孩子食物甚至穿衣等花费），从商业角度来谈，她没有理由继续抚养毛岸英哥俩。那她应该怎么做呢？

比如你把两个孩子交给我来照顾抚养，谈好每月的价钱。当你不再给我钱了，我会不会把孩子打跑？

按常理，我应该把俩孩子给你送回去，不论他是你的孩子还是你朋友的孩子甚或是毫不相干的孩子，只要给我钱我就照看。如果由于某种原因我打了你的孩子，你孩子想出走，我一定不会让他走的。我把孩子还给你时告诉你我打了你的孩子，在那年代，你不会计较。除非打残疾了打死了你才会对我报复。在幼儿园的阿姨打孩子、在学校的老师体罚学生，尤其是男孩子，是家常便饭。但我绝不会让孩子出走。我一定把孩子送回到你手里，咱们这生意就告结束，也不伤面子，更不会导致中共特务科的你对我报复而杀了我。

所以，说黄慧光打了毛岸青，毛岸英就领着弟弟出走了，黄慧光没拦着，不合常理。是瞎掰的故事骗鬼的。由于此故事太离谱，鬼很可能都不信。

那我们就退一步来看结局是怎样的。

毛岸英毛岸青哥俩从小在杨开慧身边长大。不论是毛泽东还是杨开慧，其家族都不是穷人。毛岸英哥俩从没有当乞丐的经历，他们也不知道当乞丐是否能活得下来。

毛岸英生于1922年，到1935年时已经差不多13岁的样子。现在13岁的孩子在上初二。他非常清楚是他二叔毛泽民把他们兄弟交给董健吾的。他们从大同幼稚园搬到董健吾家后和董健吾的孩子们一起过了一年半的美好时光。当他们得知黄慧光拿不到钱无法继续抚养他们后被黄慧光打了，那他们第一反应就是要黄慧光把他俩送回到董叔叔（也许是别的称呼，比如王叔叔）家去。他们在董健吾家过了一年半的美好生活不说，他们知道是董健吾答应毛泽民保护他们的。

那如果黄慧光就是不送他们回到董健吾家呢？

且说这是不合常情不合逻辑的。退十步讲，就算这是真的发生了，那毛岸英哥俩也照样能找到董健吾的家。他们在董健吾家待了一年半，他们俩是毛泽东杨开慧的孩子，不应该是傻到连家庭住址门牌号都不知道的人。根据找到的杨永寿从苏联回国（他们在苏联学的是俄语，没进过中文学校）去东北搞土改时，杨永寿（当时他就用的这个名字）不仅改正了当地领导的错误，还舞文弄墨编写土改宣传材料呢。更别说哥哥了，一定有中文基础。就算是白丁，那也知道他住了一年半的董健吾家住址。

当他俩无法在黄慧光家待下去了，必然要求黄慧光把他们送回到董健吾家。如果黄慧光不送，他俩一出门就会打听到去董健吾家的路。比如董健吾家住南京路888号（在这里具体是哪个街区哪个数字无关紧要，只是假设），他们会上海话，问问老大爷老奶奶到南京路怎么走，即使问了几个人都是冷漠自私的人不告诉他们，总会有好心人能碰上的。那他们就会找到董健吾家，对于13岁的孩子来说，打听一个在上海的住址或者说回到原来的家，应该是非常容易的。他们怎么也不会想到去乞讨的。因为他们从未有过乞讨的经历。他们脑子里第一想到的是回到董健吾家。

退一百步讲，如果董健吾不收留他们，那他们也会跟董健吾谈：“是你答应我叔叔毛泽民要对我们负责的。你不能抚养我们了，那你把我们交给我叔叔毛泽民吧。如果董健吾说找不到你叔叔毛泽民了，那他们哥俩也会跟董健吾说：把我们送回到我外婆那里也行！

道理很简单：孩子们在遇到走投无路时的第一反应是找亲人。当黄慧光不能继续抚养他俩后，他俩必然想到回到董健吾那里。毛泽民当初肯定告诉孩子们：这位（董健吾）叔叔就是你们的亲人，他会照料你们，但你们可要听话！

所以，不论是黄慧光把他俩送回到董健吾那里还是他们自己找回到董健吾那里，他们都不会选择当乞丐。当董健吾告诉他们找不到毛泽民时，他们想到的是另一亲人---外婆。他们在外婆那里度过了美好时光是无疑的，杨开慧是有教养的人，她妈妈也差不到哪里去。何况孩子的妈妈被杀了，外婆看在杨开慧的面子上也会精心疼爱他们的。他们必然思念外婆，也就会跟董健吾讨价还价要求回到外婆那里。

退一千步讲，黄慧光不送他们回到董健吾家（这不合常理）、他们找到董健吾家后董健吾不收留也不帮忙把他们转移到另一地方也不答应把他们送回到湖南外婆那里（更不合常理），这俩孩子就真的出走了，那他们一定是一边乞讨一边赶路回湖南外婆那里。只有知道世界上一个亲人都没有了的情况下，孩子才会选择靠乞讨度日一辈子。他们俩长期留在上海乞讨是没道理的，如果他们知道董健吾会找他们，他们才留在上海乞讨的话，那他们应该问路找回到董健吾家。如果董健吾搬家了，他们也会打听邻居董健吾去了哪里。他们也会常常到董健吾老家附近去等待董健吾的寻找。如果这些都无效，那他们就会从上海往湖南沿路乞讨走到老家去找外婆。要知道：在文革开始时，13岁的初中生都有步行到全国各地搞大串联的，这是我亲眼见过的，何况毛岸英兄弟--那可是天不怕地不怕的毛泽东的孩子，还是在逃脱虎口的情况下，怎么可能不逃出上海去长沙？在那时的上海，13岁打童工的很多，毛岸英宁肯打童工也不会当乞丐，而且在大街上很容易被军统和警察发现。刘胡兰14岁都领导农民打死国民党村官了。说13岁的毛泽东的儿子沿街乞讨“给口吃的吧，大伯大妈！”是对毛泽东的侮辱。

（3）黄慧光是怎么知道毛岸英哥俩去乞讨流浪了的？

按照人的思维来看，黄慧光打了毛岸青后导致毛岸英拉着弟弟离开了黄慧光家，黄慧光应该清楚他俩是回董健吾家去了。如果黄慧光是人，是有血有肉的人，而非神，那天她就不可能判断出他俩去当乞丐了。只有当董健吾去找她时她会吃惊地说：“他们走了，应该回你那里了啊。怎么？他们迷路了？走丢了？”这样的反应才对。她不可能准确得知他们乞讨流浪去了。

退一万步讲，当董健吾得知俩孩子去找董健吾时董健吾搬家了，那董健吾也不会准确判断出他俩一直在上海当乞丐，极有可能是一边当乞丐一边问路走回老家去。因为在上海不仅有生命危险而且当乞丐的日子也没有个头。他们哥俩对“上海是虎口”有亲身经历：“那时，全家承担着很大的风险，过着提心吊胆的生活，一有动静，就得外出避风头。黄慧光家多次迁移住地。从凤阳路修德里搬到成都北路3号，再搬到牯岭路斯盛里10号。有一次因情况紧急，连家具杂物都顾不上拿走。拿点替换的衣服就撤离了”（董霞飞：《特科奇人董健吾》）。

所以，董健吾清楚，要找他们，也不能一直只在上海找。在上海找不到，就得派人到从上海至长沙的路上去追。

退十万步讲，毛泽东的身高超过男人的平均水平，那毛岸英的身高在13周岁时应该超过平均水平的1米6，应该在一米六与一米七之间。如果他真的带着比他小一岁的弟弟出走了，那他也是谎报两岁说是十五六岁加入红军当红小鬼杀国民党去为妈妈报仇，而非去当可怜兮兮的乞丐过猪狗不如的低三下四的流浪生活，那是杨永福杨永寿干的事。龙生龙凤生凤老鼠的儿子会打洞。你让岳飞的儿子岳云去当乞丐？他一定拿起刀枪上阵杀敌。这是基因决定的，何况杨开慧也是宁死不屈的女汉子。毛泽东与杨开慧的儿子会不当英雄为母报仇而是沿街乞讨为了口吃的见人就喊爹叫娘求爷爷告奶奶当乞丐窝囊废？毛粉们，醒醒吧！中央特科的人骗得了明白人一时，骗得了毛粉们一世，骗不了所有的人永远。

小结：黄慧光无钱抚养毛岸英哥俩后，理应把他俩送回到董健吾手中。即使她不送，也没必要把俩孩子打跑，那将无法令董健吾等特科人息怒，甚至有被共产党杀死的危险。给人家看孩子属于生意性质，没钱就送回去天经地义，而把人家打跑是匪夷所思。即使打跑了，他们哥俩也会打听到董健吾的家。即使董健吾搬家了，他们在上海当乞丐的时间也不会太久就会往外婆那里边打听边乞讨边赶路。何况在老家毛家也有不少人。留在上海有多危险，他们是清楚的，常常跟着黄慧光躲避。说黄慧光交代说她打了岸青他俩就当乞丐去了，是对黄慧光的侮辱。只要是人，她一定会想到俩孩子被打跑后回到董健吾那里去了。当董健吾得知俩孩子出走了，在上海找一阵子抄不到，便会猜测到他俩一边乞讨一边走回长沙去了，他就会派人去沿路追赶。

所以，真实的毛岸英毛岸青哥俩到底是怎么活不见人死不见尸的，是在黄慧光手里丢失的还是在董健吾手里丢失的？是该案件最大的疑问。

如果是在黄慧光手里没了的，那绝对不是出走当乞丐去了，因为黄慧光不可能知道他俩去当乞丐的。她一定说：“是他俩回到董健吾家去了，后来发生了什么事，我不知道。”以解脱自己的责任以防被共产党杀死她。如果在黄慧光家丢失的结论是董健吾说出来的，那表明是谎言。一个解释是在给黄慧光栽赃，另一解释是被黄慧光出卖了而不能说出实情。除了这两个解释外，还能有其它的解释吗？

如果是在董健吾的手里丢失的，那我们就极难预测到底发生了什么。

我在第一篇里谈的“可能性”是黄慧光出卖了他们哥俩。其前提是：没钱了，黄慧光不把俩孩子送回到董健吾家，她还知道俩孩子去当乞丐了！这表明俩孩子的确是在她黄慧光手上没了的。

如果说黄慧光从未这么交代过，这说法来自于董健吾，那黄慧光出卖了俩孩子的“可能性”就很低了，那俩孩子在黄慧光手里没了的结论不合情理。所以，毛岸英毛岸青哥俩丢失的过程，其里边的水还很深很深，到目前为止还深不见底。我们只能根据现有的资料给出“可能性”推理，但前提是：这些资料是黄慧光提供的。如果是董健吾提供的，则另当别论。

（二十七）在毛泽东眼里，毛岸英毛岸青失踪的责任人是谁？

李云说特科找毛岸英兄弟从1935年开始是有道理的。她在这件事发生的时间上说谎是没必要的。我们可以回顾一下大的画面。

杨开慧的妈妈之所以要把三个儿童外甥送往别处，是因为有人为她通风报信了。不论在农村还是城市，如果一家人的口碑尚好，平白无故把有三个孩子的母亲给枪毙了后还要抓孩子斩草除根，一定会遭到很多人厌恶的。戴笠得知军阀何健杀了毛泽东的妻子而放走了三个儿子后派人追查三个孩子的下落，调查此事的人也许良心未泯而故意把消息透露给当地国民党官员，意思是赶跑了为止，或者当地国民党官员虽然知道他们三人的下落也故意欺骗军统人员说立刻派人去查，而事实上通风报信。杨开慧的妈妈得到军统在找毛泽东的三个儿子的消息后立刻行动。然而，毛泽东应该猜测到，军统的人可能早打听到了三个孩子在上海，以什么样的名义动手、如何能嫁祸于共产党内毛泽东的对手，是戴笠一直没动手的原因，毕竟董健吾是在国民党内有很多人脉的人物。顾顺章叛变后他都没被抓。

多疑的毛泽东还会认为是共产党内被他干掉的人比如王明在特科里就有拥趸（特科的康生在苏联时就是王明的狗，后来才改成追随毛泽东的），毛岸英毛岸青的失踪可能是共产党内毛泽东的政敌手下人为了出口恶气（我杀不死你但可杀死你孩子让你断子绝孙）而下的毒手。从时间上看，1935年1月毛泽东在遵义会议上夺得了军队指挥权，在9月份就下令撤销中央特科。这跟李云说的1935年中开始寻找毛岸英哥俩的时间吻合。当毛泽东的俩孩子丢失后，中央特科有人电报给了毛泽东。毛泽东当即判断是他的政敌手下人干的，便立刻下令解散中央特科。然而，到底是中央特科里的谁干的，毛泽东无暇顾及。他还在长征的半路上呢。

当毛泽东到了陕北，又与张国焘进入了你死我活的权力之争。他派他信得过的书呆子冯雪峰去上海找寻他儿子，很快就收到了冯雪峰的信，冯雪峰找到了俩孩子，只是未经毛泽东允许就把他们送到了苏联令毛泽东起疑到底找到的是不是冒牌货。好在冯雪峰是毛泽东的挚友，不属于特科，也就基本上认同了找到的俩孩子是真的。

然而，毛泽东必然自问：为何中央特科的人找不到，而冯雪峰一去就找到了（那时毛泽东并不知道是特科的李云找到的）？到底特科里是谁在阻扰找到俩儿子？

毛泽东想到了把俩儿子送往戴笠军统大本营的上海，是毛泽民干的。毛泽民为何把仨侄子送往虎口而他自己却在之前就逃离上海去了天津开印刷厂？毛泽东对特科和整个情治系统一窍不通，而毛泽民则一直在上海特科，跟上海特科的人熟悉得很。毛泽东得知特科创始人第二位的顾顺章叛变后没有命令毛泽民去上海搭救孩子，因为他清楚毛泽民对特科了如指掌，必然明白保护侄子责任重如泰山。可就是毛泽东的一疏忽，酿成了孩子们丢失了的大祸。此时毛泽东在苏区正遭受批判呢，他哪里顾得上孩子的事？而毛泽民则不同，他没受到批判，何况三个孩子是他毛泽民负责送到上海特科人手里的，他就应该负责到底。那为何毛泽民不去把侄子们从虎口里抢出来？读古书的毛泽东想到了另外一人---贺子珍。

当毛泽东掌权后得出“特科找不到而冯雪峰一到上海就立刻找到了”的结论便是特科的人在忽悠他，他必然要报复。是否断子绝孙放一边，拿毛泽东不当回事是绝对不能容忍的。国民党戴笠那边，毛泽东暂时还惹不起。共产党王明张国焘等政敌在特科里的爪牙则需要来个整风镇压一下。然而，当毛泽东想到是毛泽民把仨孩子送入虎口，顾顺章叛变后孩子在虎口有多危险而不前去搭救，这不是亲叔叔应该干的。他必然想到毛泽民后面有贺子珍的因素。

沉浸在历史宫廷内斗书中的毛泽东虽然有了战胜张国焘后的得意，但每当他想到特科的人对他在上海的孩子们在顾顺章叛变后未被转移，便耿耿于怀。事实上，周恩来康生等特科主要负责人在顾顺章叛变后都还在上海，周恩来亲自带康生等人杀了顾顺章老婆孩子小姨子丈母娘等9口后还一直隐藏在上海，直到1931年年底1932年年初才离开上海去苏区。在去苏区时周恩来也没把毛泽东的孩子们一同带走。这道理毛泽东明白，那时毛泽东其地位还在周恩来下面，但即便如此，周恩来也应该指示特科人员把战友毛泽东的孩子们转移出虎口（最了解上海特科的是周恩来，其次是顾顺章。顾顺章叛变后的危险有多大，可想而知。）。那周恩来为何没这么做？他猜想孩子们是毛泽民安排的，毛泽东已经有了新老婆，有了后妈后的孩子，外人不能随便插手。从这方面讲，毛泽东也开始琢磨贺子珍对杨开慧生的孩子是否排斥。有一点是肯定的：如果在得知顾顺章叛变后贺子珍当即拉上毛泽民扑去上海，那毛岸英兄弟就可能不遭受流浪之苦而后被冯雪峰找到（毛泽东会庆幸有冯雪峰这样的挚友，关键时刻能帮他的忙）。以后也就没有贺子珍的悲惨故事了。

贺子珍对毛泽东的爱是深深的。而男女之爱越深，越是嫉妒丈夫的其他女人，也越不能容忍她人与自己分享丈夫；而江青对毛泽东则是走出了男女之爱上升到“崇拜者与神人”的关系了，毛泽东喜欢张玉凤孟锦云，江青就特别尊重张玉凤孟锦云。江青让贺子珍的女儿用自己的李姓与李讷成为一对亲生女。在江青被审判时李敏也无法抱怨江青一句，因为江青对她的呵护与亲生女无异。江青对毛远新更是视为亲生儿子一般而导致华国锋孤注一掷一举粉碎四人帮。

在中国的传统文化里，是民族英雄还是卖国贼，最关键的标准是扩大了疆土还是出卖了疆土。毛泽东本人也承认“秦皇汉武唐宗宋祖成吉思汗”的历史地位。秦始皇是秦国的国君,把疆土扩大到整个华夏。我读小学时，我们的老师只在民国时念过四年私塾，他给我们讲解毛泽东诗词里的“秦皇汉武”时说是指打下天下的开国皇帝秦始皇与刘邦。几年后我才明白他讲错了，汉武指的是刘彻。刘彻为何在地位上超过打下天下的刘邦?因为刘邦没有扩大疆土，就连中原地带的河南都在匈奴的统治下，他刘邦一筹莫展。而刘彻不仅收回河南，还把西域的疆土扩大到河西走廊，后来还占领了海南岛以及越南。如果毛泽东不认同以扩大疆土为历史伟人的标准，那他就会说成是“秦皇汉祖”以打天下的刘邦代替继位的重孙子刘彻了。隋文帝虽然也大量扩大疆土，那只不过是收回了八王之乱国家四分五裂后丢失的部分。而唐太宗打下天下后就以扩大疆土为己任，虽然在高句丽那里没占到多少便宜，他也算尽力而为了，但他还是平定了吐谷浑，为了扩大疆土他一直在征战。宋祖赵匡胤的权力是从周世宗那里黄袍加身夺过来的，但他在世时把疆土扩大三倍多。一代天骄成吉思汗就更别提了，扩大疆土是他的天赋，把疆域扩大到东欧。

反过来，历史上出卖疆土的卖国贼当属后晋高祖石敬瑭。为了讨好辽国，他竟然主动把燕云十六州送给辽国。按照现在的算法，燕云十六州等于大约12万平方公里。

而毛泽东主动出让领土给朝鲜等国，可以与之媲美的只有石敬瑭一人耳!可石敬瑭也是出于对辽国的恐惧。相比之下辽国太强大了。而毛泽东把疆土白白送给朝鲜、缅甸、印度、尼泊尔、巴基斯坦等邻国则是在彭德怀打败了美国等联合国军后中国作为大国没有任何军事压力条件下出卖领土的卖国贼行为。这便成了近代史学家们眼里横竖解释不了的难题。今天，润涛阎告诉你这里边的猫腻：毛泽东为何甘当出卖领土的卖国贼?

这里需要指出毛泽东为何想借刀杀掉彭德怀。

早在毛泽东计算出打下天下没有彭德怀毫无问题时就想干掉他了，只是他命太硬而已。天下打下来后，彭德怀更成了毛泽东的眼中钉肉中刺。这事不能全怪毛泽东。刚刚打下天下，巴结毛泽东的下属们就在西山盖了五个别墅，给当时的五组（毛、刘、周、朱、任）。五个院子都一样那就显示不出来毛泽东高高在上，便给毛泽东修了个游泳池。彭德怀得知此事后当即在中央政治局会议上问毛泽东，用人民的血汗钱给你老毛修私人游泳池，这是怎么回事？在场的所有的人都敢怒不敢言，没人敢招惹彭德怀，那是因为共产党一直宣传是为了人民的利益而奋斗的，无数的烈士用献血建立的人民共和国政权，怎么变成了跟国民党一样？而且个个都是想当皇帝的奴才巴结毛泽东还怕巴结不上呢，跟刚正不阿的彭德怀相比必然自惭形秽，便对彭德怀从内心里佩服。毛泽东当即说就用他的稿费还上这笔钱。可彭德怀内心是无法满意的，你毛泽东是唯一能拿稿费的共产党领导，还有，谁知道你的稿费标准是怎么定的？建游泳池花了十万，手下人说只花了一万，我彭德怀如何去查账？

整个共产党都在巴结毛泽东，唯独彭德怀一人在监督毛泽东是否腐败，那彭德怀就是跟所有的共产党员过不去，是地地道道的反党分子。共产党高层奴才们想杀了他的心都有的不是少数。还有就是毛泽东把文工团的美女们招到中南海跳舞被彭德怀当众指责老毛是在搞后宫。毛泽东如果不杀了这个炸刺的彭德怀，那他的个人权威就受到没完没了的挑战。可彭德怀是有政治手腕的人物，不是毛泽东想杀就能杀的，属于胆大心细之人。毛泽东清楚，派彭德怀去大西北调虎离山也只是权宜之计。就在毛泽东无计可施之际，金日成与斯大林密谋侵犯南朝鲜的战役打响了，斯大林要毛泽东出兵支援朝鲜。毛泽东这才找到了“杀死彭德怀、杀死假太子、杀死几十万计的国民党降卒并给林彪的四野瘦身、让斯大林对毛泽东放心”的“一箭四雕”之计而决定出兵朝鲜所谓的抗美援朝。

当彭德怀为了制止金日成不停地出卖情报给美国情报部门而导致志司搬到哪里美国侦察机轰炸机就跟到哪里（搬家六次都遭到轰炸，九次轰炸每次都算成功炸毁了目标）不得不扇了金日成耳光。这耳光打醒了金日成。老谋深算的金日成忍下了这屈辱，他绝不轻易报复刚愎自用的彭德怀，因为他在猜测彭德怀为何要扇他耳光，背后一定有深层原因。

一九五七年，彭德怀跟随毛泽东去了苏联。金日成立刻起身也去了苏联，在他的别墅里邀请彭德怀一叙友情。彭德怀跟毛泽东怎么说的、用什么招数骗过了毛泽东至今没有披露，只知道彭德怀让毛泽东同意了他个人单独密会金日成。在两个多小时的密谈中，彭德怀得知毛泽东出兵朝鲜的一箭四雕，而金日成也确定了自己的猜测：彭德怀临走前毛泽东跟他密谈时就安排好了让彭德怀不怀疑毛泽东的电报是引发美国侦察机的元凶反而对金日成心存芥蒂。金日成这只老狐狸便等着嫉恶如仇的彭德怀报复毛泽东而痛下杀手，他太看低了彭德怀的政治水平了，以为彭德怀就是一个粗莽的武夫。彭德怀回国后不再跟毛泽东过不去了，反而在大跃进期间不进谏言，忍气吞声。他在寻找机会，潜伏爪牙忍受。他需要利用毛泽东的力量先干掉刘少奇。彭德怀想什么，毛泽东内心里是清楚的。他越是打从跟金日成密谈后宽容毛泽东，毛泽东越担心他在韬光养晦。直到庐山会议后期，彭德怀看到毛泽东还没把大跃进饿死人的罪过让刘少奇顶缸，才给毛泽东写私人信件提醒他现在饿死人太多了，你应该把罪责推给执行你毛泽东路线的刘少奇身上。毛泽东将计就计，当即联合刘少奇周恩来，先干掉你彭德怀再说别的。

这个结局令金日成非常失望。他决定要亲自出马，让毛泽东付出代价，而且是毛泽东明明知道是苍蝇也得咽下去的代价：那就是让毛泽东当中国历史上最大的出卖疆土的卖国贼。而金日成获得长白山以及天池的一半领地，就是朝鲜人包括南边的韩国人也都会崇拜金日成这个民族英雄。

长白山这个名称是辽代金代开始采用的。1712年，大清与朝鲜在长白山南麓的分水岭确定界碑，长白山主峰及天池皆在清朝版图内。朝鲜古地图《大东舆地图》中，定界碑及图们江源都在天池以南，也说明了长白山天池在中国境内，后面的中华民国延续了此界限。金日成居然提出把长白山的一角和8个山峰中的3个划给朝鲜，著名的白头山天池现为一家一半。

从庐山会议彭德怀被打倒后，金日成便派密使会见毛泽东，跟毛泽东谈长白山领土之事。金日成的密信必然只有毛泽东一人能读到，毛泽东读后便会当即烧掉。密信的内容很简单，就是提醒毛泽东你利用抗美援朝的名义一箭四雕想把杀死彭德怀的罪责推给我金日成这事得有个说法，你要是不给我领土，我就在全世界宣传你毛泽东杀死假太子并想一并杀死彭德怀，你毛泽东仔细想想中共高层尤其是军方将领得知你想借刀杀人杀死彭德怀，他们以后会怎么自己保护自己。

毛泽东左右为难，此时想赖账已经来不及了，金日成从跟彭德怀的密谈中获得了证据，那是毛泽东本人亲自告诉彭德怀的。庐山会议结束时，毛泽东还是只是罢了彭德怀的官而已，夺了他的军权以防不测，但金日成当即要挟毛泽东让他当出卖领土的卖国贼，毛泽东此时才跟刘少奇商量，提出彭德怀的最大罪恶是里通外国。刘少奇误以为毛泽东所说的彭德怀里通外国是指庐山会议前的一九五九年彭德怀去了苏联等东欧共产主义阵营国家访问时出卖了中国的利益。刘少奇建议据此判处彭德怀死刑。随着刘少奇的栽赃，跟班们越说越离谱，说彭德怀私自与赫鲁晓夫密谈。彭德怀至死不承认他会俄语，他从未秘密会见过赫鲁晓夫。而毛泽东所说的彭德怀里通外国是指他与金日成密谈两个多小时时暴露了毛泽东跟他谈论的去朝鲜后不要听金日成的调遣还要防金日成出卖彭德怀住处的情报给美国大兵。然而，这事毛泽东在庐山会议后绝不能跟任何人说明真相，他只能指鹿为马含糊其辞地说彭德怀里通外国，具体是什么，毛泽东只字不提。

然而，即使杀了彭德怀，毛泽东也还是满足不了金日成的要求，因为金日成要的是领土，而非彭德怀的命。

金日成反复派密使找毛泽东要领土，毛泽东不答应是非常危险的，金日成有苏联的靠山就不怕毛泽东。而毛泽东清楚，只要金日成说出真相，毛太子是假的，被毛泽东借刀杀人给烧死了，彭德怀九次被美国轰炸机轰炸，都是毛泽东没完没了给彭德怀打电报彭德怀不得不回复电报惹的祸。这样的披露一旦经过边境电台播放，很快就会被中共上层得知。大家立刻醍醐灌顶，彻底明白了找到的毛岸英毛岸青是冒牌货，对将军们来说毛泽东想杀死彭德怀的事远比杀死假太子重要得多。毛泽东承受不了这样的揭秘，便答应要给金日成一部分领土。

然而，这事麻烦大了。

中共高层人人都会琢磨：为何金日成胆敢骑在毛泽东头上拉屎撒尿？我们欠金日成什么了？抗美援朝我们死了几十万人，美国还没从南朝鲜撤走，金日成就让志愿军统统离开北朝鲜。在抗美援朝过程中苏联出的武器弹药钱，凭什么要中国还债？是苏联答应金日成进犯南韩的，是苏联让中国出兵的，凭什么我们出人命他们出枪炮到头来我们算借钱？要还，那也应该是北朝鲜还才对，参加抗美援朝战争是帮助金日成的江山，我们中国人为何当冤大头？毛泽东有何短处被金日成给抓到了竟然赔了太子又折兵还要把疆土送给他？

（二十九）毛泽东批判苏联修正主义的心理根源

毛泽东冥思苦想想到了万全之策：背叛列宁主义而拥抱马克思主义。

此时的毛泽东清楚，他是惹不起金日成的，金日成光脚而他却穿着鞋。毛泽东思前想后找到了糊弄手下人的办法：背叛列宁主义、拥抱马克思主义。这样就可以出卖疆土了。

毛泽东在北大时就听李大钊介绍明白了为何共产主义理论里有了马克思主义还来个列宁主义？

马克思死了35年后列宁才革命成功，并提出了列宁主义理论。列宁主义跟马克思主义是完全不同的东西。如果马克思多活35年，那他绝对不允许列宁欺世盗名把列宁那一套说成是什么发展了的马克思主义。

列宁主义的本质特征有几点是与马克思主义背道而驰的：

1.马克思主义认为，资本主义高度发达后导致的先进生产力与落后的私有制生产关系的矛盾无法调节，才走向公有制的共产主义（共产主义分阶段，社会主义是共产主义的第一阶段，但社会主义阶段也一定是在高度发达的资本主义之后才能实现）。

而列宁主义刚好相反，在生产力极其低下并未经过资本主义高度发达的国家就可以“穷过渡”到社会主义和共产主义。

2. 马克思主义认为，在从资本主义进入共产主义的无产阶级暴力革命阶段，全世界无产者联合起来，工人阶级没有祖国，只有先解放全人类才能最后解放无产阶级自己。

而列宁主义刚好相反，共产主义首先在一个国家取得政权，工人阶级不仅有祖国，还要进行爱国主义教育，培养爱国主义热情。然后扩大共产主义阵营，向其它国家输出革命，最后实现共产主义大同世界。
不仅仅是毛泽东，所有的中国共产党高层领袖们历来都是学苏联的理论与经验，最早建立的苏区全称便是“中华苏维埃政权”。毛泽东打下天下后提出的是以苏联为老大哥，并把马列主义作为口号。是金日成令毛泽东不得不改弦更张，毛泽东出卖领土后便只字不提 “马列”，代之以“马克思主义”。处处提的都是“要搞马克思主义”。

道理很简单：根据马克思主义的本质原理，毛泽东可以把任何中国的领土送给其它国家也不是卖国贼，而是马克思主义者。而根据列宁主义的本质原理，本国的领土是不能出卖的，只能是输出革命，让其它国家的共产党打下江山，巩固其领土不受侵犯。

最早修正马克思主义的就是列宁。列宁主义地地道道地修正了马克思主义，简称为修正主义。所以，毛泽东后来就骂坚持列宁主义的苏联共产党为修正主义，简称苏修。其苏修的鼻祖就是列宁。列宁的理论与马克思的理论背道而驰，只是在暴力革命方面与马克思主义一致。马克思主义的精髓在于资本主义高速发展了的先进生产力与落后的生产关系矛盾无法调和才搞公有制,是生产力决定生产关系。而列宁主义刚好反过来，要用先进的生产关系公有制反过来拉动落后的生产力。这是马克思绝对不认同的东西。所以，毛泽东只能利用列宁主义的模式夺取政权，然后搞“穷过渡”，大跃进失败后他也没有认同先搞资本主义，待资本主义高度发展了生产力后再过渡到公有制的“二次革命论”。当金日成要挟他必须把领土白白送给朝鲜否则就暴露他的“一箭四雕”借刀杀人想杀死彭德怀的肮脏行为，他才不得不在理论上找根据以糊弄过去。

理论上有马克思主义做挡箭牌，但他出卖领土给北朝鲜还是很容易受到他手下人的质疑，他便想到了另一策略：先把领土白白送给缅甸。他派刘少奇跟缅甸谈，在1960年就把江心坡7万平方公里的领土白白送给了缅甸。然后他才派周恩来与金日成派来的代表谈判，先是把鸭绿江改为中朝共同拥有的河界，把中国独自建好了的鸭绿江上游的云峰发电站发电量一半无偿白输给朝鲜。金日成当然不会就此罢休，他要为他在朝鲜历史上建立丰功伟业，最后还是派代表要求亲自和毛泽东密谈。密谈的内容世人无法得知，但最终毛泽东同意把长白山一分为二。毛泽东的官方说法是：“金日成提出长白山是金日成的革命事业的发祥地，属于金日成的革命圣地，应让朝鲜人民世代瞻仰。”云云。但这些说法是糊弄傻子的。金日成说什么毛泽东就得言听计从？按照这样的胡言乱语逻辑，那毛泽东的一个儿子还是在莫斯科出生的呢，如果此男孩没夭折，那毛泽东就应该跟苏联提出莫斯科属于中国因为那是他儿子的出生地不成？

毛泽东把领土出卖给金日成后，虽然在理论上有“要搞马克思主义”作支撑，在实践上先给缅甸领土为烟幕弹，他还是担心手下人不好糊弄，更不敢把出卖领土的事让全国人民知道（如果不是金日成要挟，而是出于他对马克思主义的信仰，那他就会大力宣传他是虔诚的马克思主义者而把领土送给邻国），他随后把藏南9万平方公里领土送给印度（1962年）和大片领土白白送给阿富汗与巴基斯坦（1963年）。去谈判的分别是他手下权力最大的刘少奇与周恩来。这样，其他人就不敢指责他是卖国贼了，因为除了彭德怀外，所有的高层领导都是毛、刘、周的人（就是朱德也是周恩来拉拢进共产党的，南昌起义失败后朱德逃到井冈山依附了毛泽东）。

毛泽东出卖领土给朝鲜的过程，网上有介绍，是从1959年年底开始金日成派人到中国要领土，毛泽东采取拖延战术，先出卖领土给其它国家，然后拖到1962年才结束与朝鲜的谈判。下面一段摘自网络：

1962年毛泽东指派周恩来总理访问朝鲜，代表中国政府同代表朝鲜政府首相金日成在平壤签订了《中朝边界条约》。此条约共有五条，主要是第一条，划分了两国边界的走向。座落于中朝边界的长白山天池和白头峰，历来是我国的神圣领土。天池位于长白山之巅，乃火山爆发铸成的九峰围合而成，最高一座名白头峰。历史上中朝国界线在分水岭东下20公里处，自南向北划定。即使在日本统治朝鲜时期，天池也在中国版图；朝鲜建国时对此也是承认的。如今毛泽东大手一挥就把天池切了一半（一说百分之五十三）给朝鲜，分水岭东侧的三座山峰也捎带送了出去。朝鲜立即将白头峰改名“将军峰”。

引用毕。

毛泽东从此便以“要全党搞马克思主义”为口号从心理上给自己当上了卖国贼而壮胆。中国历史上，出卖领土的皇帝都是在打不过敌人的条件下不得不为之。就是晚清把台湾出卖给日本，也是甲午战败后的无奈之举。历史上未被外敌打败就主动出卖领土者，只有后晋皇帝石敬瑭，而出卖领土的面积远比不上毛泽东，也只是出卖给了辽国，而辽国后来又被灭了，燕云十六州现在是中国的首都所在地。而毛泽东出卖的领土全部要回来还不知道多少年后。所以，毛泽东自己清楚他“一箭四雕”的妙计结果是人算不如天算，彭德怀未被杀死不说，还成就了彭德怀历史上民族大英雄的地位，而自己则是历史上主动出卖领土的最大卖国贼。历史档案俱在，铁证如山。

那么，有没有可能是润涛阎冤枉了毛泽东？芦笛网友曾经写过一个长文论证了毛泽东出卖领土的原因，题目就是《我党背叛马列，使伟大领袖成了卖国贼》（大意如此）。据润涛阎考证，芦笛文章里面的事实是可靠的，毛泽东成为卖国贼是真实的，但原因则不可靠。

毛泽东从来都不是马克思主义者，他最多是主张“穷过渡”的列宁主义者。毛泽东天天看的书不是马克思的资本论，而是线装书帝王之术。据他自己说，《资治通鉴》他就读了17遍。他恨透了郭沫若的骂秦始皇的文章，并说他自己是马克思加秦始皇。那他为何不只承认自己是当代秦始皇，而非要把马克思膏药也贴身上？因为根据秦始皇的理念，毛泽东出卖领土也照样是卖国贼。唯独把马克思（无产阶级没有祖国、只有解放全人类才能最后解放无产阶级自己、消灭国家）那一套皮毛套在秦始皇身上才能洗刷掉他在历史上卖国贼罪名。而对马克思的“资本主义高度发展了的生产力与私有制生产关系不适应才建立公有制生产关系”这一马克思主义的核心内容，他毛泽东是背道而驰的。

从时间上看，恰恰是彭德怀被打倒后金日成便要挟毛泽东。如果没有彭德怀跟金日成密谈的事，金日成断不敢轻举妄动要挟毛泽东，尤其是让毛泽东这个读线装书的皇帝当卖国贼，不是金日成能轻易办得到的。金日成掌握了毛泽东“一箭四雕”杀死了假太子试图杀死彭德怀的阴谋诡计，毛泽东清楚这事一被揭穿，手下的将军们个个毛骨悚然，一有机会就会玩命自保的。毛泽东不得不跳下金日成给他挖的“卖国贼”坑，反正他是“断子灭孙”（毛泽东原话）了，管它死后怎么评价，只要大权在手他的罪恶不被手下人揭发出来就行。

打从毛泽东不得不出卖领土当上了卖国贼后，他内心恐惧得很，被迫害妄想症越来越严重。据汪东兴等人的材料披露，毛泽东不仅不敢长期在北京生活，就是在外地，也是不按他自己同意了的预先安排行动，突然间专列就停下来在半路上过夜。今天怀疑这个要夺他的权，明天那个要杀他，他就没完没了地整人。其实他内心里是自保。说起来这也不能全怪毛泽东，他的俩儿子哪里去了都查不出来不说，还被俩乞丐给骗惨了，连挚友文人冯雪峰都靠不住。他疑神疑鬼是有原因的，俗话说一朝被蛇咬十年怕井绳。

两年前全国人民都在议论薄熙来的一个耳光把王立军打到美领馆去了，把自己打入牢房，这耳光作用太大了。其实，与彭德怀的那个耳光相比，薄熙来扇的耳光简直不值一提。彭德怀扇政治家老狐狸金日成的那个耳光，不仅把毛泽东打成了中国历史上最大卖国贼，还把大量领土打到了中国周围的国家去了。

正因为彭德怀一耳光导致中国大片领土丢失的惨痛结局，彭德怀在他的自述里只字不提这件事。他只是对刘少奇指责他“里通外国”审查他时问及他是否与赫鲁晓夫私自密谈，他才矢口否认他曾“里通外国”，而他心里非常清楚毛泽东告诉刘少奇说他里通外国只是给个罪名而毛泽东不能告诉刘少奇是彭德怀跟金日成密谈时把毛泽东告诉彭德怀的话出卖给了金日成。毛泽东不敢说，彭德怀更不想说，毕竟这大好河山的领土白白送给了外国与他彭德怀有牵连。如果他能预料到这样的结局，那他绝对不会去密会金日成了。

彭德怀为了当民族大英雄，命都不顾；毛泽东为了个人权力，当历史上的卖国贼都不在乎。人品高下立判。可见，权力的魔力有多大，能让人心甘情愿当历史上的卖国贼。

为了不让全体中国人民清楚他出卖领土的行为属于卖国贼性质，他不让人民读古书。不论你读什么古书，哪怕是小说，都有可能明白出卖领土是卖国贼的传统文化。可见他对当上了历史上的卖国贼是何等害怕。只是为了权力，这些也顾不上了而已。

小结：

当金日成得知彭德怀陪同毛泽东去了苏联，便立刻起身去苏联“度假”，他不与下令志愿军赴朝救了他的江山的毛泽东会面，而专门找扇了他耳光的彭德怀密谈。密谈中确定了毛泽东为达“一箭四雕”成功借刀杀人的目的而不得不告诉彭德怀当心金日成出卖情报给美国，等于给金日成栽赃，导致彭德怀扇了金日成耳光。老狐狸金日成看穿了毛泽东的把戏，在美国大兵不撤出南朝鲜的情况下当即让中国军队彻底撤出北朝鲜，毛泽东全盘接受后，金日成便清楚彭德怀扇他耳光的背后有毛泽东给金日成栽赃陷害的把戏，否则毛泽东不会那么软，便找机会与彭德怀密谈。确定真相后，金日成等于握住了毛泽东的把柄。当彭德怀被毛泽东干掉后金日成得知靠彭德怀干掉毛泽东的梦泡汤了便要挟毛泽东，让毛泽东送领土给他。

如果有人认为金日成是个疯子，对恩人毛泽东不仅不报恩还耍无赖，跟毛泽东要领土，是毛泽东倒霉碰上了个疯子邻居而已。假如这个假设是真的，金日成就是一个不知天高地厚的疯子，那他达到目的还需要另一条件：毛泽东必须是一个对疯子金日成有求必应并自己甘愿当卖国贼的傻子。可事实是，金日成既不是疯子，毛泽东也不是傻子。

毛泽东用“要搞马克思主义”的口号为他当出卖领土的卖国贼找理论依据，但他并不敢把出卖领土的事告诉人民大众，表明他心虚。为了糊弄手下人，他得出不得不给金日成领土的结论后，为了掩盖被金日成要挟的真相，他先把7万平方公里的领土给缅甸，然后再送给金日成半个长白山和天池的领土，他此时还是害怕手下人猜测到真相，便当即把更多中国领土送给印度阿富汗尼泊尔巴基斯坦越南等国。毛泽东用马克思主义的大旗盖住了他想借美国轰炸机杀掉彭德怀反被金日成报复的结局，的确蒙骗了差不多三代人总数以十亿为单位，但他蒙骗不了政治家们肚子里的蛔虫润涛阎。

因为润涛阎不相信文韬武略的金日成是疯子（对救命恩人毛泽东不感恩戴德也算了还跟他要领土）也不相信读线装书的毛泽东是甘愿当卖国贼的傻子（马克思的资本论毛泽东一遍都不读而资治通鉴他读了17遍。历史上什么是卖国贼，他清楚得很）。而只有金日成是疯子毛泽东是傻子才能发生这等匪夷所思的甘愿出卖领土历史故事，否则必有不可告人的内幕。

金日成也好玩得很。他在文革期间来北京见毛泽东之前下令把假太子的墓给砸了，然后到北京跟毛泽东热烈拥抱。他清楚那是假太子的墓。如果是真太子，了解中国文化的金日成知道“宁扒一座庙不扒一座坟。”扒儿子的坟比扒祖宗的坟更残忍。这浅显的道理金日成明白得很。那要是真太子，他胆敢扒坟，就得做好跟毛泽东兵戎相见的准备，他还敢到北京？到北京后毛泽东还会跟他热烈拥抱？

识别毛泽东出卖疆土是因为毛泽东是马克思主义者还是被逼无奈有不可告人的原因只看一条：

如果毛泽东是马克思主义者为了实践马克思主义才出卖疆土，那他是主动的，是自己首先找邻国提出来的；如果是有不可告人的原因被逼无奈，整个过程则是被动的。拿这个标准验证一下毛泽东出卖疆土给金日成便得知，从官方的表述看，是金日成从1959年开始不停地派人到中国提出领土要求，等于毛泽东是被动的，是不得不给的，被逼无奈的，有不可告人原因的。所以，他事后用“要搞马克思主义”做借口其用意是掩盖他被逼无奈不可告人的内幕，以给共产党高层洗脑暗示他的做法根据马克思主义不属于卖国贼行为。

后记：

现在所有的共产党员都知道毛泽东出卖了大片领土给缅甸、朝鲜、印度、越南、尼泊尔、阿富汗、巴基斯坦等几乎所有与中国相邻的国家，但为何毛泽东要这么做，还没有一个人提出过令人信服的合乎逻辑的解释。毛泽东在理论上承认秦皇汉武唐宗宋祖成吉思汗是英雄，在实践上他拿下新疆西藏还派谢飞带兵为拿下台湾试水，表明他还是想当扩大疆土的英雄的，而非要当出卖领土的卖国贼。

退一步说，如果说毛泽东信了马克思主义而不把出卖领土当成卖国贼才大方地送领土给邻国，那他应该大张旗鼓地宣传把大片领土送给邻国是实践马克思主义的伟大创举。可事实上他极端害怕人民知道他出卖领土的真相。全国人民都不能用短波收音机收听境外电台广播，一经发现以现行反革命论处。如果不是杨永福杨永寿哥俩被动当了欺骗毛泽东的骗子，毛泽东即使派兵去朝鲜抗美援朝，没有假太子在彭德怀身边，那他清楚没完没了地给志愿军司令部发电报招来美国侦察机轰炸机借刀杀人的谋略很快就会被彭德怀识破。如果不想杀死彭德怀，那他绝对不会把抗美援朝的战功送给天天找他茬的彭德怀，而会直接派有大兵团作战经验的林彪去指挥。毛泽东一边让彭德怀去指挥朝鲜战争，一边让林彪做赴朝的准备，其中的道理再简单不过，那就是他在等彭德怀的死讯。毛泽东万没想到，彭德怀是杀不死的命硬分子，彭德怀的一耳光把金日成给打醒了，金日成报复毛泽东的方式非常高明：让毛泽东甘当出卖领土的卖国贼，而金日成则成了骑在毛泽东头上拉屎还让他吃掉的最大赢家，至今三代执政成功。而毛泽东则再次演绎了“偷鸡不成蚀把米”的新编寓言故事。金日成这名字毛泽东晚年想起来一定不寒而栗。

在芸芸众生眼里，毛泽东是想干什么就干什么的神人，而在润涛阎眼里，毛泽东是以骗人始以被骗终（被叫饭花子哥俩骗得无可奈何哑巴吃黄连还不得不接受被骗结局）；以玩弄人始以被人玩弄终（被金日成玩成了卖国贼）；一个可怜的断子灭孙（毛泽东原话）的老头。死后都不能入土为安，骗了他的杨永寿留下的傻儿子还打着毛孙的旗号继续骗着他、羞辱着他。
